Installation Requirements

Seagate Crystal Reports has the following installation requirements:

  • Microsoft Windows 95, Windows 98, Windows 2000, Windows NT 4.0, or higher
  • Minimum RAM: 16 MB (32 MB for Windows NT)
  • Recommended RAM: 32 MB
  • Minimum hard drive space required: 30 MB (Full Install 195 MB)
  • Maximum hard drive space required: 195 MB

Note: If your system does not meet these requirements, the program may not run correctly.

Installing Seagate Crystal Reports on a Local Machine

Seagate Crystal Reports uses an Install Expert to step you through the installation process. The Install Expert automatically recognizes the operating system on your machine and updates any files as required.

If you want to limit the features you install, see Customizing your installation.

1.    Double-click Setup. Depending on the configuration of your current system, you may see a dialog box informing you to update existing files.

2.    Click Yes and restart your machine. The Install Expert updates the required files.

3.     In the User Information dialog box, enter your name and the CD Key Code printed on the sticker on the back of the CD envelope.

4.    Click Next. The Select Installation Type dialog box appears.

5.    Select the installation type.

6.    Typical installs the most common application features. Complete installs all application features and subcomponents. Custom enables you to choose what features you want installed and where they will be installed.

7.    Choose Custom if you want to check the amount of space (disk cost) that each feature requires, install specific features, or change the default location of various features. If you select the custom option, see Customizing your installation.

8.    Click Browse to choose the destination folder to install Seagate Crystal Reports. The default is C:\Program Files\Seagate Software\Crystal Reports\.

9.    Click Next and follow the instructions in each dialog box to complete the install.

Customizing your Installation

Use the custom installation if you want to check the amount of space (disk cost) each feature requires, install specific features, or change the default location of various features.

1.    Double-click Setup. Depending on the configuration of your current system, you may see a dialog box informing you to update existing files. Click Yes and restart your machine. The Install Expert updates the required files.

2.    In the User Information dialog box, enter your name and the CD Key Code printed on the sticker on the back of the CD envelope.

3.    Click Next.

4.    In the Select Installation Type dialog box, select Custom for the installation type.

5.    Click Browse to select the destination folder to install Seagate Crystal Reports. The default is C:\Program Files\Seagate Software\Crystal Reports\.

6.    Click Next. The Select Features window appears with a list of Seagate Crystal Report features.

7.    Click the driver icon of the feature(s) you want to install.

8.    Select the configuration and location of the feature.  

9.    Click Next. Follow the instructions in each dialog box to complete the install.
