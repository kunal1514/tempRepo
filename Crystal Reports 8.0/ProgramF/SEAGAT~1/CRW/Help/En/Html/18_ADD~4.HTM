About the Microsoft Access Add-In

Choose a table or query to create a Seagate Crystal Report. The report you create is linked to your Access table or query and can be refreshed to show changes made to the database.

Note:    

  • Access Add-In reports will not reflect database changes until you close the modified table or query in Access and refresh the report data in Seagate Crystal Reports.
  • Access Add-In reports must be based on one table or query (but not a query with input parameters).

After you create a report using the Wizard, you can preview it using the default ActiveX viewer or you can launch Seagate Crystal Reports or Seagate Analysis to modify it.
