Total tab

Use the Total tab to choose fields to be subtotals, counted, and so on.

Available Fields

This list displays the report fields available for totaling in the report.

Summarized

This list displays the total fields currently in use in the report.

Add ->

Click this button to add the selected field from the Report Fields list to the Total Fields list.

<- Remove

Click this button to remove the selected field from the Total Fields list.

Browse Data

Click this button to browse the data in the field selected in the Report Fields list.

Summary Type

Click a summary function for the field selected in the Summarized list.

Percentage of

Select this check box, then click a summary field in the list to show the summary value for the current grouping as a percentage of the summary value for another grouping.

Add Grand Totals

Select this check box to include Grand Totals in a report.
