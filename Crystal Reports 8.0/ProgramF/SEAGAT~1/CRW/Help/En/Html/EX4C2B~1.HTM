Data tab

The Data tab contains three sections:

  • Placement
  • Layout
  • Data

In the Placement section, you determine where you would like your map to be placed on the report, as well as how often the map will appear on the report.

In the Layout section, you determine the actual layout of the map, depending on the data you wish to map. There are four options:

  • Advanced option view
  • Group option view
  • Cross-Tab option view
  • OLAP option view

In the Data section, you determine the data that will be mapped.

Place map

The Place Map list gives you the option to place a map only Once per report or For every instance of a group.

Header and Footer

Use the Header and Footer option buttons to determine where in the report your map will be placed.
