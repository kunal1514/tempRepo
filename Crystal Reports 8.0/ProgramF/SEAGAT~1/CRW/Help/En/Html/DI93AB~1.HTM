Change Group dialog box

This dialog box appears when you choose the Change Group Expert command from the Report menu.

Use the Change Group dialog box to reposition groups and to edit the criteria used for triggering and sorting groups.

Groups

This list lists all the groups in the report.

When you enter a group, that group appears directly above and below the Details section. When you add a second group, it appears between the Details section and the first group. The third group is between the Details section and the second group, and so on. Thus, when you look at the Design tab, the Group Header for the first group is the top group header, and the Group Footer for the first group is the bottom group footer.

This group list reflects the group footer order of groups; the bottom group in the list is the first group, the group next to the bottom is the second group, and so forth.

When you select a group in this list, the program activates those buttons that apply to the selected group.

Options

When you click this button the Change Group Options dialog box appears. Use this dialog box to edit the criteria used for triggering and sorting groups.

Move Up

Use this button to move the selected group up in the group hierarchy. For example, if you grouped first by state and then by country, the country group is at the top of the list followed by the state group. If you decide that you want your data grouped by country first and then, within each country, by state, you need to move the state group up. To do this, select the state group and click the Move Up button. When you run your report, the grouping will take place in the new order.

Move Down

Use this button to move the selected group down in the group hierarchy. For example, if you grouped first by state and then by country, the country group is at the top of the list followed by the state group. If you decide that you want your data grouped by country first and then, within each country, by state, you need to move the country group down. To do this, select the country group and click the Move Down button. When you run your report, the grouping will take place in the new order.
