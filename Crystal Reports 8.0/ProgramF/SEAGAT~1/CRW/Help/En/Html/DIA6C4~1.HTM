Remove from Report dialog box

This dialog box appears when you select the Remove from Report command from the Database menu.

Use the Remove from Report dialog box to delete databases from the active list so they can no longer be used in your report.

The alias of each active database appears in the Databases box.

The name of the selected database and its path appears in the Location box.

Databases

Lists all the active databases used in the current report.

Location

Displays the referenced table, server type, server name, database and user ID of the selected table.

Remove

Click this button to remove the selected database.
