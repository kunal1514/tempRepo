Choose Tables To Use In Visual Linking dialog box

This dialog box appears when you click the Add/Remove Tables button in the Visual Linking Expert.

Use the Choose Tables to Use in Visual Linking dialog box to choose the tables you want to link in your report using the Visual Linking Expert.

Linked Tables

This list box displays all those tables that are currently linked.

Visible Tables

Visible tables are the tables you want to use in your report. When you add a new data file to your report by clicking the Add Table button, the program lists all the tables in that data file in the Visible Tables box unless there are more tables than you might practically use in a report. In that case the tables are listed in the Invisible Tables box and you can select the tables from that box that you want to make visible.

All tables in the Visible Tables box are available for linking and appear in the Field Explorer, the Formula Editor, and so on. When you link a table from the Visible Tables box, the program moves the table name from the Visible Tables box to the Linked Tables box.

Invisible Tables

Invisible tables are the tables you do not want to use in your report. Tables listed in the Invisible Tables section will not appear in the Field Explorer, the Formula Editor, and so on.

  • To make an invisible table visible, select the table in the Invisible Box and click the Add button with the left arrow. This adds the tables to the Visible Box. You can also click the Add All button with the left arrow to add all the tables to the Visible section.
  • To make a visible table invisible, select the table in the Visible Box and click the Add button with the right arrow. This adds the tables to the Invisible Box. You can also click the Add All button with the right arrow to add all the tables to the Invisible section.
Add Table

Use the Add Table Button, to add further database tables to your report. When you click the Add Table button, the Data Explorer appears. Use this dialog box to select the database file or files you want to add to your report. Each time you select a file and click the Add button, the database file is added to the Visible Tables box unless there are more tables in the data file to practically use in a report. In this case the tables are added to the Invisible Tables box.

When you have added the database or databases you need, click Close to return to the Choose Tables To Use In Visual Linking dialog box. The database or databases you added will be in the Visible Tables box.

Perform Smart Linking

When the option is selected, Smart Linking is activated. When the option is not selected, Smart Linking is inactive.

When Smart Linking is turned on, and you click OK in the Choose Tables To Use In Visual Linking dialog box, Smart Linking identifies suggested link fields automatically and performs the links. You can verify the links performed by Smart Linking in the Visual Linking Expert.

Add Index

Click this button to add an index to the currently selected table. The dialog box that opens is a standard Windows search dialog box, called Attach New Index.

Table Description

When you click this button, the Table Description dialog box appears giving you a description of the currently selected table.

All (>>)

Click this button to add all the tables displayed in the Visible Tables list to the Invisible Table list.

Add (>)

Click this button to add the selected table in the Visible Tables list to the Invisible Table list.

Remove (<)

Click this button to remove the selected table from the Visible Tables list box and add it to the Visible Table list.

Remove All (<<)

Click this button to remove all the selected tables from the Visible Tables list box and add them to the Visible Table list.
