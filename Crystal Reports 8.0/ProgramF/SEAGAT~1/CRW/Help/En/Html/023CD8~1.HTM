Performance

Fastest performance ever

Seagate Crystal Reports 8 features major improvements to report performance in all areas of processing.

Seagate Crystal Reports has the capacity to handle larger data sets much more quickly. You will see performance improvements during your entire cycle of report creation—from opening and refreshing to formatting and saving your reports.

View reports with saved data immediately

Due to changes in accessing saved data on-demand, you can now open and view the first page of many reports immediately.

Faster processing of refreshed data

Seagate Crystal Reports is able to generate better SQL (Select Distinct, pushdown OR clauses, and more record selection) so less data is returned from the database.

Enhancements to calculations for formulas and summaries also provide faster processing of records returned from the database.

Jump to the last page

Significant improvement in page formatting enables you to see the last page of your report sooner.

Smaller file size

RPT and Temp files are generally much smaller in Seagate Crystal Reports 8 due to a new file format.

You have the flexibility to save a report to version 7 if you require backward compatibility.
