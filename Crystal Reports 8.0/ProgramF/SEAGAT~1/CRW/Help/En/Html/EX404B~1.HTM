Type tab

The Map Expert provides five basic map types, each suitable for a different strategy of data analysis. When deciding which map type best fits your report, you should consider the type of data you want to analyze. For example, if you want the map to display one data item for each geographic division (city, state, country, and so on), then you might use a Ranged, Dot Density, or Graduated map. On the other hand, if you want the map to display more than one value for each new geographic division, then you could use a Pie Chart map or a Bar Chart map.

The map types are:

  • Ranged
  • Dot Density
  • Graduated
  • Pie Chart
  • Bar Chart

There is also a separate Options section that allows you to decide the appearance of the map values when they are displayed on the map. For example, you can choose a color for the lowest interval on your map.
