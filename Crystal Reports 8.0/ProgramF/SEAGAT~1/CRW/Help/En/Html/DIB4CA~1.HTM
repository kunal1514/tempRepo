Print dialog box

This dialog box appears when you choose the Printer command from the Print menu (under the File menu), or when you click the printer icon in the Preview tab.

Use the Print dialog box to set the specifications for your print job.

All

Select the All option to print all the pages in range on the open report.

Pages

Select the Pages option to print only a selected range of pages on the open report. Use the From and To boxes to enter the range of pages.

Copies

Type the number of copies you want to print.

Collate Copies

Select Collate Copies if you would like the printer to collate the report when multiple copies are printed. This is selected by default.
