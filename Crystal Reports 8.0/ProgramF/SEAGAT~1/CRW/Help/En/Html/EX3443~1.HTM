Form Letter tab

Use the Form Letter tab to add and delete fields and set the formatting properties of a form letter report.

Section

This list displays all of the report sections currently available for use in a form letter report.

Available Fields

This list displays a list of all the database fields available for use in a form letter report.

Browse Data

Click this button to browse the data in the field selected in the Database Fields list.

Add to Text

Click this button to add the selected field in the Database Fields list to the Form Letter Fields list.

Form Letter Edit box

This list displays all the fields currently in use in a form letter.

Import Text

Click this button to import already prepared text for use in your Form Letter report. When you click this button, an Open dialog box appears where you can choose the desired text file.

Format Text

Click this button to set the formatting properties for a Form Letter report. When you click this button, the Rounding tab (Format Editor) appears where you can set paragraph, font and tab options.

Delete

Click this button to remove the selected field from the Form Letter Fields list.
