Report Viewer Demonstration

If you are interested in trying out the functionality of a Web Component Server and Report Viewer, and have access to a web server, you can explore the Seagate Software Web Samples.

To access the Web Samples
  1. Click Start, select Programs, select Seagate Crystal Reports, then click Web Samples and Utilities Page.

    The Seagate Crystal Reports Web Samples & Utilities Home page appears.

  2. In the Report Server Samples area, click view the samples.

    The Report Server Samples page appears in your default browser.

  3. Choose a server from the Select a Server Type drop-down list.
  4. Choose a Report Viewer from the Select a Viewer drop-down list.
  5. Click view sample.

    The report appears in the Report Viewer specified.
