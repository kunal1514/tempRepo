Data tab (OLAP)

Note:    The same Data tab also appears in the Format OLAP Grid dialog box and Insert OLAP Grid Object dialog box.

Use the Data tab on the OLAP Report Expert to give specific information about your OLAP data source:

  • the specific cube you intend to report on,
  • the connection to the cube,
  • the name of the database, and
  • the type of OLAP data source.

Note:    You will not be able to proceed beyond the OLAP Data tab until you have properly selected an OLAP cube. Furthermore, working with OLAP data requires a previous understanding of the OLAP concept. Make sure you are familiar with OLAP concepts and your OLAP software before attempting to design OLAP-based reports. If you have questions, please contact your System Administrator.
