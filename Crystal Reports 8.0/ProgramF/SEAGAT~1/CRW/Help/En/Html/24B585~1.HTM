Exchange Folder contents

Description

Accesses all properties for each object in a folder (including all additional properties that the Exchange form assigns).

The folder that you would like to access can be viewed by selecting the folder in the Exchange client (provided that you have the appropriate permissions). The report designer will interpret the selected folder in the following manner:

  • the folder will be seen as a database table
  • each object contained within the folder will be seen as a record of the folder table
  • each object's properties will be seen as fields of the folder table's records.

In addition to providing access to all of the default properties that Exchange assigns an object within a folder, the Report Designer will provide access to any fields that were created by an Exchange Form. If the Exchange Form is:

  • assigned to the same folder that you specified when logging on as you selected upon creating the report
  • in the Exchange Client's Personal Forms Library.

The report designer will make it's fields available for access.
