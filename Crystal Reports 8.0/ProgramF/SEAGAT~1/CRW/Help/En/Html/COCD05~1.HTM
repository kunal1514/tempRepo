Change Group Expert command

Use the Change Group Expert command to change the sorting and grouping specifications (sort and group by field, sort direction, and so on) for any of the groups in your report. You can also use this command to insert and remove groups and to change their processing order. If you insert a group section, a subtotal, or a summary and later want to modify it in any way, use this command to do so.
