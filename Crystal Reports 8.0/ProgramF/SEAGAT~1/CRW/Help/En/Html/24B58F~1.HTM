Server type

Exchange Folders/Address Book

To log on to the Exchange Folders/Address Book server type
  1. Open the Data Explorer dialog box by choosing:
      • Log On/Off Server from the File menu (when there is no report opened)
      • Log On/Off Server from the Database menu (when there is a report opened).
  2. Select Exchange Folders/Address Book from the More Data Sources folder.
  3. Double-click the folder or double-click Make a New Connection. The Choose Profile dialog box appears.
  4. Select the Profile that you would like to use from the Profile Name list. Click OK.
  5. From the Select a folder/address book dialog box, select the folder that you would like to use and click OK.
To set the location of a table accessing an Exchange Folder
  1. On the Database menu, click Set Location.
  2. When the Set Location dialog box appears, click Set Location.
  3. Find and highlight the new Exchange Folder you want to use in the Data Explorer dialog box and click Set when done.

    Verify that the new table has been selected by reviewing the Table, Server Type, Server Name, Database (alias name displayed), and User ID in the Location section of the Set Location dialog box.

  4. Click Done after reviewing the table information.
