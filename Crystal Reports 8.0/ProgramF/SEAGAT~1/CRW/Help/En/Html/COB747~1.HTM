OLE Object command

Use the OLE Object command to link or embed an OLE object in your report.

Note:    You can also insert a picture using the Picture command on the Insert menu. In that case, the Report Designer converts the picture into a static object, which you cannot edit. To convert a static object to an editable object, use the Convert Picture Object command on the Edit menu.
