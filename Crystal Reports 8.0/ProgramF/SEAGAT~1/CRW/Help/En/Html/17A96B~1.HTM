Exporting to Microsoft Mail (MAPI)

  1. Open the report you would like to export.
  1. On the File menu, select Print, then click Export.

    The Export dialog box appears.

  2. Select the export format type from the Format drop-down list. In this case, choose Excel 7.0 (XLS) (Extended).
  3. Select Microsoft Mail (MAPI) from the Destination drop-down list.
  4. Click OK.

    The Format Options dialog box appears.

  5. Change the formatting options as needed.
  6. Click OK.

    The Send Mail dialog box appears.

  7. Enter the address details, then click Send.

    The Exporting Record dialog box appears.

Note:    Click Cancel Exporting to cancel the export process.
