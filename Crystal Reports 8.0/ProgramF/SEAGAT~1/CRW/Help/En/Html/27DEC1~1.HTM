Adding graphics to the dictionary

  1. While in the Crystal Dictionary Expert, click the Graphic tab.

  2. Click the Add button.

    The File Open dialog box appears.

  3. Use the controls in the File Open dialog box to highlight a graphic image file, and click OK to add it to the dictionary.

    Dictionaries support the Windows bitmap (.gif), PC Paintbrush (.PCX), Tiff (.TIF), and TARGA (.TGA) graphic formats. The image file appears in the list box of the Graphic Tab.
