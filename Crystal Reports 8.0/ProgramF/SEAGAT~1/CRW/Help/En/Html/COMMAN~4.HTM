Tabs command

Use the Tabs command to set the tab stops for paragraphs in a text object. You can specify tab stop position and tab alignment. In order to activate this command, you have to double-click the text object.

Choosing the Tabs command opens the Text Format dialog box, with the Tabs tab selected.
