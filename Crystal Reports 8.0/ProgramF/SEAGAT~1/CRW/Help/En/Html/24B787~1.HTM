Public Folder allocation and usage

Description

Provides valuable configuration and usage information of all Public Folders in an organization.

Seagate Crystal Reports will provide information for all of the folders listed in each Exchange Server's Public Information Store (as represented by the profile(s) selected when logging on). To view this data, select each Exchange Server's Public Information Store{Public Folder Resources tab} in the Exchange Administrator to view all of the properties that the Report Designer provides. The program interprets each entry as a record and the columns (the items available in the Report Designer as fields of the record.
