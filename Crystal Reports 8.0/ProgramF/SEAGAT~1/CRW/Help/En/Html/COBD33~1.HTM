Delete Section command

Use the Delete Section command when you have multiple sections in an area and you want one of them deleted. For example, if you have three Detail sections, Detail A, Detail B, and Detail C, and you want to delete Detail B, use this command to delete that section.

This command takes you to the Section Expert, which you can use to select the section you want to delete, then click Delete to delete it.

Note:    You cannot delete any of the original sections that appear when you first create a new report.
