Selecting a query for a report

  1. From the Start menu, select Programs, select Seagate Crystal Reports, then click Crystal Report Designer.
  2. On the File menu, click New.

    The Seagate Report Gallery appears.

  3. Confirm Standard is selected, then click OK.

    The Standard Report Expert appears.

  4. Click Query, from the Data tab.

    The File Open dialog box appears.

  5. Locate the queries (.QRY) file you want to use to create the new report.

    This example uses the sample query in the Queries folder.

    Note:    Only one query file can be used with each report.

  6. Click Open.

    The query file for your report is added to the Data tab.

  7. Continue working through each of the tabs in the Standard Report Expert to finish designing the report.
