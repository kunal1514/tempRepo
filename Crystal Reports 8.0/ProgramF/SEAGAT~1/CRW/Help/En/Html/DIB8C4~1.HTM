Technical Support Request dialog box

This dialog box appears when you choose the Technical Support Request command from the Help menu.

Use the Technical Support Request dialog box to request assistance with Seagate Crystal Reports from the technical support department.

You can fill out the technical support request form, print it out, and then FAX it or mail it to the technical support department for a timely response.
