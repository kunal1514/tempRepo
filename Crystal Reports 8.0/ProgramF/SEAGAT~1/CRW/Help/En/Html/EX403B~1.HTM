Advanced option view

Use the Advanced option view when you have multiple map values, rather than only one, or when you do not have any groups or summaries in the report. With the Advanced Layout you determine which fields are used for the Geographic breakdown, the On change of value, and the Map value. You can also set summary options.

Note:    You must map on a geographic field, such as Country, Region, or City. The program cannot generate a map based on a non-geographic field.

Available Fields

The Available Fields list allows you to select which fields to use as the data for your map.

Browse

Click the Browse button to view the data contained in the selected field in the Available fields list.

Add

Click the Add button (>) to copy fields from the Available fields list to the Geographic Field text box, the On change of text box, or the Map values list.

Remove

Click the Remove button (<) to remove a selected field from the Map values list.

Remove All

Click the Remove All button (<<) to remove all fields from the Map values list.

Geographic field

Use the Geographic field text box to determine the geographic breakdown of the map. For example, if you insert the field Customer.Country into the Geographic field text box, your map will display values for each country. If you insert the field Customer.Region, your map will display values for each region.

On change of

Use the on change of list to specify a field to use as a condition for displaying a summary field. Summary values will be displayed whenever the value in the condition field changes.

Map values

Use the Map values list to create a list of all the fields you want shown as values on the map.

Don't summarize values

Use the Don't summarize values check box to determine whether the program should automatically summarize the values that will appear on the map.

  • If you select this check box, the program will not summarize the values.
  • If you clear this check box, the program will summarize the map values, and the Set Summary Operation button will activate. The program adds a summary field to the Map Values list.
Set Summary Operation

Use this button to set or change the summary operation for the selected field. When you click this button, the Change Summary Operation dialog box appears. This button is only active when you have cleared the Don't summarize values check box, and when you have selected a field in the Map Values list.
