Change Geographic Map dialog box

The Change Geographic Map dialog box opens when you choose the Change Geographic Map command from the Analyzer menu, or from the map's right-click menu. Change Geographic Map is available only when you are working with a map on the Analyzer tab.

Use this dialog box to specify the map you'd like to see if you want to have your data values presented with a different geographic map. For example, if the data used on your map applies only to Asia, you can choose the map of Asia instead of the default World map. This change gives you a more detailed map of Asia for a better presentation of your data.

Current Map

This box displays (for viewing) the geographic map that is currently used for the map selected on the Analyzer tab. The same geographic map is highlighted in the Candidate Maps box.

Candidate Maps

This box displays a list of the possible geographic maps you can select for your report. Highlight the geographic map of your choice and click OK, or double-click the Candidate Map itself. Your map on the Analyzer tab changes to the appropriate geographic map.
