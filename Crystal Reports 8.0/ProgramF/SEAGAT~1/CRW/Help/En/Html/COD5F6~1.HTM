Close All command

Use the Close All command to close all open windows. Seagate Crystal Reports gives you an opportunity to save any unsaved work in the open windows.

When you choose the Close All command:

  • if you have saved all your reports, the program closes all the windows.
  • if you have not saved all your reports, the program prompts you to save them, report by report.
      • Click Yes to save the report before closing the window.
      • Click No to close the window without saving the report.
