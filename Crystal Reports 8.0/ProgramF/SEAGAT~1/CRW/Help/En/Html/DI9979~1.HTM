Choose OLAP Type dialog box

The Choose OLAP Type dialog box allows you to choose the type of OLAP data source to work with in your report. This dialog box appears when you click Server Type on the Data tab of the OLAP Report Expert.

When working with OLAP grid objects, the Choose OLAP Type dialog box contains the following OLAP types for you to select from:

  • Essbase Cube
  • IBM DB2 Cube
  • OLE DB for OLAP
  • Informix MetaCube
  • Holos HDC Cube.

Each of these types, except the last, require you to connect to a server. The last type, the Holos HDC Cube, requires you to enter the path for an HDC cube location.

Connect To Servers

This dialog box opens when you choose Essbase Cube, IBM DB2 Cube, or Informix MetaCube. Enter the OLAP Server name and your user ID and password to connect to the server.

Connect to OLAP DB Servers

This dialog box opens when you choose OLE DB for OLAP. Enter the OLE DB Provider name, the OLAP Server name, your user ID and password, and any parameters required to connect to the server.

Browse for Database

This dialog box opens Holos when you choose Holos HDC Cube. Enter the path and name of your HDC cube, or click Browse. Browse opens the Add a Holos default file dialog box so you can search for your HDC cube.
