Baan

Baan, a vendor of client/server business application solutions, offers Baan IV, an ERP (enterprise resource planning) software package. Baan IV includes a custom SQL client/server and database you can report on using Seagate Crystal Reports.

Seagate Crystal Reports provides access to databases created by Baan IV ERP. The P2sbaan.dll file installed with Seagate Crystal Reports translates requests to the Baan SQL server and then returns data from the server.

Note:    

  • The Baan client does not have to be installed to use the Seagate Crystal Reports translation file.
  • Seagate Crystal Reports supports access to the Baan database using the files Bn4c2nt2.dll and P2sbann.ini.
