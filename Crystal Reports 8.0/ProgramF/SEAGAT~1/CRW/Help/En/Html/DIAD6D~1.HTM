Format Line tab (Format Editor)

This tab appears in the Format Editor when you right-click a line and choose the Format Line command from the shortcut menu. You can also select this option from the Format menu.

Use the Format Line tab to set the formatting specification for a line in your report.

Style

This list shows all the available styles for your line, such as: single, double, dashed, and so on.

Width options

Click the button that represents the line width you want to use.

Color

This list shows all the available colors for your line.

Move to bottom of Section when Printing

This option is typically selected when you are printing a line next to a field that may produce variable length output (a memo field, for example) and you format that field using the Can Grow (Common tab/Format Editor) option. In this case, if the program needs to grow the section to accommodate field output, it will grow the line to the bottom of the section as well.

  • Select this check box if you want the line to always extend to the bottom of the section regardless of the length of the section.
  • Clear the check box if you want the line to be the same length in each section, regardless of section size.
Suppress

Select this check box if you do not want the line to print.
