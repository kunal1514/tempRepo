Form tab

Use the Form tab to format your form and add bitmaps.

Section

This list displays a list of all the report sections currently available for use in your report.

Add Picture

Click this button to add a bitmap to the section chosen in the Sections list. The bitmap will be previewed in the preview window.

Delete Picture

Click this button to delete the bitmap previewed in the preview window.

Preview Window

This window shows a preview of the contents of the currently selected section in the Sections list.
