Subreport command

Use the Subreport command to create a subreport object that you can place on your report. A subreport object is a self-contained report that has all of the usual report characteristics and abilities. You can import an existing report to use as a subreport object, or you can create your subreport from scratch as you would any other report.

You can insert multiple subreport objects into your parent report if you wish. You can coordinate the subreport data with the data in the primary report, or you can leave your subreport freestanding and unrelated to any data in the parent report.

When you choose the Subreport command, the Insert Subreport dialog box appears.

Note:    This command is also available as a button on the Standard toolbar.
