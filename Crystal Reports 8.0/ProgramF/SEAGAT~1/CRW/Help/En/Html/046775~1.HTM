Saving the report

  1. Click Save on the Standard toolbar to save your work. Since this is the first time you are saving the report, the Save As dialog box appears displaying the default directory where the file will be saved.
  2. Type Custlist.rpt in the File name box and click Save. Your report is saved to the default directory or another directory you chose.
  3. Click the Preview tab to view the report.
