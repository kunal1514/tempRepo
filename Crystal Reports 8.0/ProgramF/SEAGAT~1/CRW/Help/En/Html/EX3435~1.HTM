Filter/Page tab (OLAP)

The Filter/Page tab allows you to choose what to do with other dimensions that are not rows/columns in the grid. The Filter list displays all of the dimensions you did not use in the Rows/Columns tab. The Filter/Page tab lets you specify how dimensions that do not appear in the grid will affect the data for the dimensions that do appear in the grid.

Filter

Choose one field from each of the dimensions. The dimensions will be set to these fields in the grid.

Move (< >)

Click these buttons to move selected dimensions between the Filter and Page lists.

Order

Click the up and down arrow buttons to change the order of the Filter and Page dimensions.

Page

Choose multiple fields from each of the dimensions. A separate grid will be created for each field.

Select

Use this button select the fields you want displayed for a filter or page dimension.

When you click this button, the Select a Field for Dimension dialog box appears.

The field you select for a filter dimension appears to the right of that dimension.
