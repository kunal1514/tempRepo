Subtotal command

Use the Subtotal command to insert a subtotal of the currently selected numeric field into your report. A subtotal is a summary that can only be used with numeric fields. The program sorts, groups, and subtotals in a single step. When you choose the Subtotal command, the Insert Subtotal dialog box appears.
