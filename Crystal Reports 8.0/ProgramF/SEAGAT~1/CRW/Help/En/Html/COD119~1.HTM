Technical Support Request command

Use the Technical Support Request command to fill out a technical support request. When you are finished, you can print the report and either fax or mail it. When you choose this command, the Technical Support Request dialog box appears. Enter the information in this dialog box.

Note:    You must send in the registration form in order to receive technical support on your product.
