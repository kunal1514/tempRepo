Using a web browser to view reports

Your Systems Administrator or Information Systems Manager will have set up an intranet/extranet site where you can view your reports in a browser. Contact your web administrator to ask for the URL. This URL will specify both the Web Component Server and Report Viewer.

There are three types of Web Component Servers:

  • WCS via ISAPI\NSAPI Extension
  • WCS CGI Extension
  • ASP Reports Server

There are six types of Crystal Report Viewers:

  • Report Viewer for ActiveX
  • Report Viewer for Java using Browser JVM
  • Report Viewer for Java using Java Plug-in
  • Report Viewer for Netscape Plug-in
  • Report Viewer for Standard HTML with Frames
  • Report Viewer for Standard HTML

Using the appropriate Web Component Server and Report Viewer, you can easily view and print your reports from a web browser.

Note:    Crystal Report Viewers for Java and HTML can be used on any platform, while the Report Viewer Plug-in and the Report Viewer for ActiveX are made specifically for Windows environments.
