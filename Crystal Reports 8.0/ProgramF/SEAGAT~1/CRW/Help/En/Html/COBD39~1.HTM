Merge Section command

Use the Merge Section command when you have multiple sections in an area and you want two of the sections combined into a single section. For example, if you have three Detail sections, Detail A, Detail B, and Detail C, and you want to combine Detail B and Detail C into a single section, use this command to merge them.

This command takes you to the Section Expert, in which you select the first of the sections that you want to merge, then click Merge to merge that section and the next. The sections are renumbered automatically.
