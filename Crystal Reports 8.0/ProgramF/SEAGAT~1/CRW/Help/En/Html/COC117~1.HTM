Section command

Use the Section command to display the Section Expert. Using the Section Expert you can:

  • insert, delete, merge, and move report sections,
  • hide a section (keeping it from printing) while retaining drill down capabilities,
  • suppress a section (keeping it from printing and eliminating drill down capabilities),
  • print subtotals or group values only at the bottom of the page,
  • insert a page break before the section is printed,
  • insert a page break after the section is printed,
  • reset the page number to one (1) after a group value prints,
  • prevent page breaks from spreading data from a single record over two pages,
  • prevent blank lines from printing,
  • cause an object to overlay the following section when it prints, and
  • format the section with multi-column capabilities.
