Inserting fractional font sizes

On the Font tab of the Format Editor, you can select a fractional font size for database fields and text-based objects on your report.

To insert fractional font sizes
  1. Right-click the field or object you want to format to bring up the shortcut menu.
  2. On the shortcut menu, click Change Font.

    The Format Editor dialog box appears with the Font tab open.

  3. In the Size list, select a fractional font size for the field or object.
  4. Click OK to save your changes.

Note:    You cannot select a fractional font size for every report you work with by selecting the options on the File > Options > Fonts tab. You can only select a fractional font size for individual database fields and text-based objects.
