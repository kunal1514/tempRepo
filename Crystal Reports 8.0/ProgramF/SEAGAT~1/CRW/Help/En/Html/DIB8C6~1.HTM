Table Description dialog box

The Table Description dialog box is primarily a read-only dialog box that provides information on any table displayed in either the Visual Linking Expert or the Choose Tables To Use In Visual Linking dialog box.

  • To call up the dialog box from the Visual Linking Expert, double-click on any field in the table or right-click the field and then choose Description from the shortcut menu.
  • To call up the dialog box from the Choose Tables To Use In Visual Linking dialog box, select the table of interest and click the Table Description button.

Use the Table Description dialog box to get detailed information on the fields and indexes in the selected table.

Indexes section.

This section contains two lists and an Add and a Delete button.

  • The box on the left indicates all the indexes that the program has identified as being connected to the selected table. You can select any index in this box by clicking the index.
  • The box on the right indicates those fields that are components of the index currently selected in the box on the left.
Fields section

This section describes the following about a selected field:

  • if the field is part of an index
  • the field description
  • the field data type and length.

The fields section contains two lists and a Browse button.

  • The box on the left indicates all the fields in the selected table. You can select any field in this box by clicking on the field name.
  • The box on the right gives a description of the field and indicates if it is a component of any index.
  • When you click the Browse button, the program calls up a dialog box that indicates the name and source of the selected field, the field type, and the field length. It also displays a list of values in the selected field.
