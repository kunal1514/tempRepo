Paste Special dialog box

This dialog box appears when you choose the Paste Special command from the Edit menu.

Use the Paste Special dialog box to paste an OLE object into your report from the clipboard.

Paste

Click this to embed the copied information into your report.

Changes you make to the embedded object will not be reflected in the original file, and changes to the original file will not be reflected in this report.

Paste Link

Click this to link the copied information to your report.

Changes you make to the embedded object will be reflected in the original file, and changes to the original file will also appear in this report.

As

Enables you to select how the copied information is pasted into your report. This option is not available for all types of object. See OLE for details.

Display As Icon

Specifies whether the embedded object will appear in the report as an icon or whether the actual contents of the object will always be visible. Note that some object applications only display objects as icons.

If you select Display as Icon, you can open the object in your document by double-clicking its icon.

Change Icon

Click this button to change the icon that represents the embedded object in your report.
