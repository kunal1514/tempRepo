Mapping concepts

Mapping overview

With Seagate Crystal Reports, you can include geographic maps in reports. Maps help you analyze report data and identify trends more efficiently. For example, you could create a map that shows sales by region. You would then be able to:

  • use one of the five map types to analyze the data
  • use the Analyzer tab to adjust the appearance and organization of the map (allowing you to better identify trends)
  • drill down on the map regions to view underlying data.

In order to place a generic, group-based map on a report, you must first have a group and a summary or subtotal field present for that group. However, the program does provide several specialized map layouts that do not require groups and summaries.

Note:    When a field is summarized or subtotaled, the program automatically groups the data. For more information, see Sorting, Grouping, and Totaling.
