Display Properties dialog box

The Display Properties dialog box appears when you click the Display button on the Layer Control dialog box. The Display Properties dialog box allows you to further manipulate the mode and Zoom layering of a selected layer.

Display within Zoom Range

By clearing this check box, the selected layer is shown at all zoom levels of the map. By selecting this check box, the selected layer will only be shown within the range set in the Min Zoom and Max Zoom boxes.

Min Zoom

Use this box to set a minimum zoom level for the selected layer.

Max Zoom

Use this box to set a maximum zoom level for the selected layer.
