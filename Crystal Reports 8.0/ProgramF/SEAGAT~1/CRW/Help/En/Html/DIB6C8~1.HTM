Set Alias dialog box

This dialog box appears when you choose the Set Alias command from the Database menu. Use the Set Alias dialog box to select an alias that you want to change.

Databases

List the aliases used in the current report. The first alias on the list is selected.

Location

This information box at the bottom of the dialog box displays the referenced table, server type, server name, database and user ID of the selected alias.

Set Alias

Opens the Alias Name dialog box, in which you can change the alias of the selected database.
