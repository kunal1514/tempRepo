Using the Tabs command

Use the Tabs command to set the tab position and alignment for paragraphs in a text object.

  1. Double-click the text object you want to format and choose the Tabs command from the Format menu. The Text Format dialog box appears with the Tabs tab (Text Format) active.
  2. Enter the desired tab position into the position edit box.
  3. Specify the desired tab alignment by choosing the appropriate option button.
  4. Click the Set button to set the tab.
  5. Repeat steps 2 through 4 to set additional tab positions.
  6. Click OK to return to your report.

Note:    The unit of measurement used in the dialog box (inches or centimeters) is based on the Regional setting in the Windows Control Panel (for example English(United States) = inches, English(United Kingdom) = Centimeters).

Note:    You can also set indentation, margins, and tabs by double-clicking the text object and using the in-place ruler.
