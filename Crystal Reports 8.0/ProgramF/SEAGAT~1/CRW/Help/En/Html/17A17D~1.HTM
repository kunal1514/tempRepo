Using the Report Viewer for Java (Java plug-in)

  1. Type the URL for the report into your web browser. Obtain this from your Systems Administrator or Information Systems Manager.

    The first page of the requested report appears inside the Report Viewer. A Smart Navigation Group Tree is generated based on the groups in the report.

  2. Expand the tree and click a group to navigate to a specific part of the report.

Note:    You may not have access to Smart Navigation with the Group Tree. Smart Navigation may be disabled for the report or for the Report Viewer itself.

In addition to the functions provided on the Standard toolbar, this Report Viewer has the following extra functions in its customized toolbar:

  • Close current view
  • Go to first page
  • Go to previous page
  • Specify page number
  • Go to next page
  • Go to last page
  • Stop
  • Refresh
  • Show/hide group tree
  • Magnification factor
  • Text to search for
  • Find next.

When you place the mouse pointer over a button on the customized toolbar the associated ToolTip appears in the bottom left side of the screen.
