Preview Sample dialog box

This dialog box appears when you choose the Preview Sample command from the Print menu (under the File menu).

Use the Preview Sample dialog box to select how many records you want to preview.

All Records

Select this option to preview all the records available in the report.

First n records

Select this option to preview a specified number of records available in the report.
