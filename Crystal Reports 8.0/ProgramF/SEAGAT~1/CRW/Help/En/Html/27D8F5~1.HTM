Linking multiple tables

If you have added more than one database table to your dictionary click the Links tab in the Crystal Dictionary Expert.

Creating a link

  • If a link is possible between two tables, you can create a new link by dragging a field name from one table to the other. The application will draw a new link arrow between the tables.
  • If you click the Smart Linking button, the application will create logical links between tables in your Dictionary.

Changing a link

  • If you select a link and click the Options button, the Link Options dialog box appears. You can use the controls in this dialog box to make any necessary changes to the highlighted link.
