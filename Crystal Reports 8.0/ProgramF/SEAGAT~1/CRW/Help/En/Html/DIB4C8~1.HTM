Print Setup dialog box

This dialog box appears when you choose the Printer Setup command from the File menu.

Use the Print Setup dialog box to select the printer, page orientation, and paper size you want to use to print the report. If you do not select a printer, the program will print to the Windows default printer.

No Printer

Select the No Printer check box when printing is not an option from a particular workstation.

Name

Find the name of the printer to which you can send reports in the Name list.

Properties

Click the Properties button to bring up the Properties dialog box for the selected printer.

Default Properties

Select the Default Properties check box to set all printer properties to their default settings.

Size

Use the Size list to select from a variety of paper- or envelope-size options.

Source

Use the Source list to select from a variety of feed or paper source options.

Orientation options

Select the Portrait option to print your report in a standard-letter orientation.

Select the Landscape option to print your report in landscape orientation.

Network

Click the Network button to connect to a network printer.
