Overview of the Seagate Crystal Reports Add-Ins

Seagate Crystal Reports takes advantage of Microsoft's Add-In technology by offering a report creation Wizard you can use with Microsoft Excel and Microsoft Access. The Seagate Crystal Report Wizard offers familiar report design capabilities directly within your Microsoft Office applications—you can create a Crystal report from your data without leaving Excel or Access.

The method for working with the Add-Ins is similar in both Excel and Access:

  • Install the Seagate Crystal Report Wizard in your Office application (only if you installed Office after you installed Seagate Crystal Reports) by using the Microsoft Add-In Manager.

    You need to do this only once. The Add-In remains a part of the Office application until you choose to remove it.

  • Select a spreadsheet (or selected cells), table, or query and launch the Wizard.
  • Use the creation Wizard's familiar screen layout to create a report to your specifications.

Note:    The Add-Ins work with only the Office 97 and Office 2000 versions of Excel and Access.
