Database layer

The database file which the ODBC data source refers to can be located anywhere on a system. Once the ODBC data source is set up, Seagate Crystal Reports does not need to know the actual location and format of the data. Thus, the database can have any format and can be located anywhere on a network, as long as ODBC can communicate with it through the ODBC data source.
