Selecting groups to be included in a query

  1. Select the field from the Database Fields you want to summarize.

  2. From the Total drop-down list select the aggregate function to apply to the field.
  3. Click the Select tab.

  4. Select the aggregate function from the Database Fields area.
  5. Click Add.

    The field is added to the Select Fields list.

  6. Use the selection criteria drop-down lists to specify which group summary values based on the aggregate function should appear in the result.
