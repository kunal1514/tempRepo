Database Fields

Use the Database Fields branch of the tree to add database fields to your report. Fields are listed by table.

If there are many active tables, or if the number of fields in a table is very large, only the table name for each of the tables appears. When you click the + sign in front of the table, or double-click the table name itself, the program displays the fields in that table. Double-click the table name again and the program displays only the table name.

You insert database fields by dragging them into the report, or by selecting them and then clicking Insert to Report. You can select multiple fields using the Shift-click or Ctrl-click combinations.

Each field added to the report has a check mark added in front of its name in the Field Explorer.

Insert to Report

Click this button to insert the selected field(s) in your report. When you do, the program displays a placement frame. Move that frame where you want it on the report and click to place it.

Browse

Click this button to browse the data in the selected field.
