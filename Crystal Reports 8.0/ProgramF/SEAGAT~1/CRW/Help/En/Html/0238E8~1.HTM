Integration with Microsoft Office

These enhancements are available in all editions of Seagate Crystal Reports.

Create reports from Microsoft Excel spreadsheet data

Using the Add-in for Excel, create reports using data from a Microsoft Excel spreadsheet. You have the option to choose and limit the Excel data you want available in the report. To maintain consistency with the data, any data changes in Excel are updated automatically in Seagate Crystal Reports. Compatible with Excel 97/2000.

Create reports from Microsoft Access data

Using the Add-in for Access, create reports from Microsoft Access queries or tables. The report remains linked to your Access database table or query so any data changes in Access will be displayed when you refresh your report.
