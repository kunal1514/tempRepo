Set Default Values dialog box

Use the Set Default Values dialog box to specify the default prompts you want to appear with your parameter field. This dialog box appears when you click the Set default values button in the Create Parameter Field dialog box or the Edit Parameter Field dialog box.

Browse table

Use this list to pick the table you want to browse for fields/values to use for changing the Default Values for your parameter field.

Browse field

Use this list to pick the field you want to browse for values to use for changing the Default Values for your parameter field.

Select or enter value to add

Use the Select or enter value to add box to enter values in the Default Values list. You can type new values in the Select or enter value to add box and then press the add button (>), or you can select a value from the Select or enter value to add list (which will then be placed in the Select or enter value to add box) and then press the add button (>).

Add (>)

Use the add button to move values from the Select or enter value to add box to the Default Values list.

Add all (>>)

Use the add all button to move all values in the Select or enter value to add list to the Default Values list.

Delete (<)

Use the delete button to remove a value from the Default Values list.

Delete all (<<)

Use the delete all button to remove all values from the Default Values list.

Order arrow buttons

Use the Order arrow buttons to move a value up or down in the Default Values list. For example, if you created a value for a new employee, Zoe, and you would like the value Zoe to appear as the default value for the parameter field you are creating. You could select the value Zoe in the Default values list and continue to click that value until it was located at the top of the list, before, for example, the value Alex.

Default Values

Use this list to create a list of default values for the parameter field you are creating. The items you choose for the Default Values list may represent a subset of the items you chose in the Browse Table list and the Browse Field list. For example, if you selected {Customer.REGION} from the Browse Field list, perhaps you might designate AL, CA, and OR as your default values. To add those three values to the Default Values list, you would select the values in the Select or enter values to add list, and then you would click the add button to add the selected values to the Default Values list.

Import pick list

Click this button to import a text file containing a pick list of parameter values.

Export pick list

Click this button to export the pick list of parameter values to a text file.

Define description

Use this option to add a description to any value in the Default Values list. Clicking this button opens the Define Description dialog box.

Length limit

By selecting the Length limit check box for a String value, you can determine the minimum or maximum length for a parameter value entered on the Enter Parameter Values dialog box. This option is not selected by default.

Min Length

Use the Min Length box to limit the minimum number of characters that can appear in your String value parameter field. For example, if you choose the number 4 as your minimum length limit, the name of customer Dan could not be used because it is fewer than 4 characters long.

Max Length

Use the Max Length box to limit the maximum number of characters that can appear in your String value parameter field. For example, if you choose the number 5 as your maximum length limit, the name of customer Margaret could not be used because it is more than 5 characters long.

Edit mask

For a string parameter field, you can choose to enter an Edit Mask in the Edit Mask box, rather than specifying a range. The Edit Mask can be any of a set of masking characters used to restrict the values you can enter as parameter values (the edit mask also limits the values you can enter as default prompting values). For more information on using the Edit Mask, see To define entry type and format using the Edit Mask.

Note:    When you enter an Edit Mask, the length options become inactive.

Display

Use the Display list to select how you'd like the Default Values in the Enter Parameter Values dialog box to appear. If you select Value and description, the default values are displayed as the values and their corresponding Descriptions. If you select Description, only the descriptions of the values are displayed.

Order

In the Order list, you can specify whether you want the default values sorted. The sorting order can be alphabetically ascending, alphabetically descending, numerically ascending, numerically descending, DateTime ascending, or DateTime descending.

Order based on

In the Order based on list, you can specify whether to sort default values based on value or description.

Note:    If you apply numerical sorting to an alphabetic value or description, the default values will not be sorted.
