To underlay a chart

  1. Create your chart and place it in the Report Header section. See Creating charts.
  2. On the Insert menu, click Section.

    The Section Expert dialog box appears.

  3. In the Sections area, click Report Header, then select the Underlay Following Sections check box.
  4. Click OK.

    Seagate Crystal Reports returns you to the report. Your chart will now underlay the sections below it.

  5. If necessary, move or resize the chart.
