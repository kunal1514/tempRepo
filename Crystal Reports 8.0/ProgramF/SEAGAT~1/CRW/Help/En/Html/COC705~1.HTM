Using the Subtotal command

  1. Select the field that you want to subtotal.
  2. Choose the Subtotal command from the Insert menu. The Insert Subtotal dialog box appears.
  3. Use this dialog box to set the conditions that trigger the printing of a subtotal.
  4. Click OK. The program automatically inserts the subtotal into the group section of your report. If you have already entered other subtotals for the same field, the program creates a new section for the new subtotal.
