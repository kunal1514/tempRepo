Launch Analyzer command

Use the Launch Analyzer command to activate the Analyzer tab for a selected chart or map in the Preview tab.

The Analyzer tab enables you to organize your charts and maps for more efficient analysis. This tab also enables you set the properties of an individual instance of a chart or map independently from the rest of the report.

Note:    

  • On the Map shortcut menu, this command is called Map Analyzer.
  • On the Chart shortcut menu, this command is called Chart Analyzer.
