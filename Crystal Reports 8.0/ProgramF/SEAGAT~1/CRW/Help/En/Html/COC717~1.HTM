Using the Copy command

  1. Select the text or object you wish to copy.
  2. Choose the Copy command from the Edit menu. The program makes a copy of the selected text or object and sends it to the Clipboard for later use (or until it is overwritten when you next use the cut or copy command). The text or object you selected in the report remains in place, unchanged.
