Cross-Tab Report Expert

The Cross-Tab Report Expert guides you through the creation of a report in which your data is displayed as a cross-tab object. Three special tabs (Cross-Tab, Style, and Customize Style) help you create and format the cross-tab itself.

The Cross-Tab Report Expert has the following tabs:

  • Data tab
  • Cross-Tab tab
  • Style tab
  • Customize Style tab (OLAP and Cross-Tab)
  • Chart tab
  • Select tab
