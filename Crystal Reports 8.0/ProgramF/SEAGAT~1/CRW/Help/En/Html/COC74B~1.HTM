Border and Colors command

Use the Border and Colors command to add borders, background fill, and drop shadows to fields on your report. For example, you can:

  • enclose the field in a box,
  • enclose the field in a partial box or frame (only the top, only the left side, the top and bottom together with no sides, and so on),
  • add drop shadows that print below and to the right of the field value,
  • change the color of the border, text, and add a background fill color to a field,
  • specify the border width (the full width allotted for the field, or the width of the data within the field), and the border height (the full height allotted for the field, or the height of the data within the field).

Using the border options, you can create a variety of striking effects that can enhance the look of your report and highlight important data.
