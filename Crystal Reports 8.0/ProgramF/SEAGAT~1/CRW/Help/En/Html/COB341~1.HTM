SQL Expression Field command

Use the SQL Expression Field command to insert an SQL Expression in your report. You must have some familiarity with the SQL language and SQL expressions before trying to add an expression to your report.

When you choose the SQL Expression Field command, the Field Explorer appears with Running Total Fields selected.

Note:    You can also open the Field Explorer by clicking the Insert Fields button on the Standard toolbar.
