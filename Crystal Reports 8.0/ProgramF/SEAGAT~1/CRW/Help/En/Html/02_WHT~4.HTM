Enhanced charting and mapping

Create charts and maps based on running totals and OLAP grids

Create charts and maps using a variety of data, including running totals and OLAP grids.

Define and control the zoom area of charts and maps

You can now define the area of a map, line chart, or bar chart on which you want to zoom. Use the zoom features to examine the specified area in greater or less detail.

Always know where you are with the Map Navigator

When you are zooming in and out of maps, the Map Navigator shows the entire map and highlights the portion of the map you are currently viewing. Use the Map Navigator to pan from one area to another.

Compare multiple totals in one chart

Use a graphic representation to compare totals such as last year's sales versus this year's sales or actuals versus budgeted.

Create maps based on unique groups

Let your specified groups define the borders of a map. If you have a report that groups data based on unique geographic groups, you can see a visual representation of these groupings.

Resolve mismatched data with map locations

View the data that does not match the geographic names in the map. For example, a map of North America may use "Unites States of America" while the data uses "USA." Use the Map Region Resolver dialog box to view and add data that was not in included in the map.
