Choose Export File dialog box

This dialog box appears when you click the OK button in the Number and Date Format dialog box which you can open by choosing the Export command from the Print menu (under the File menu).

Use the Choose Export File dialog box to specify the file name (and path) to which you want to export the file.

  • Using the Save in list, specify the folder in which you want to save the file.
  • In the File name box, specify the file name you want to use.
  • Click Save. The program exports the data to the file you specified.
