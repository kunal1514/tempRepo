Working with text objects

You will use many of the capabilities of text objects when creating form letters. A brief discussion of text objects should make it easier for you to create the form letter in the next section. Some things to consider are:

  • a text object can contain both text and fields; you will use both in this example
  • you can resize text objects; you will be resizing the text object so that it prints as a letter.

Text objects operate in two modes:

  • the move/resize mode
  • the edit mode.
