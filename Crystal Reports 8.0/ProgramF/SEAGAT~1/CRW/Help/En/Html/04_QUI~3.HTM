Creating the report

  1. Click New on the Standard toolbar.

    The Seagate Crystal Report Gallery appears.

    The gallery contains a number of experts to guide you through the creation of specific types of reports. Since you will be learning reporting concepts here, you can skip the experts and build your report from scratch. After you have completed this tutorial, you may want to build some reports using the experts to decide which method of report construction you are most comfortable with.

  2. Select As a Blank Report on the Seagate Crystal Report Gallery dialog box and click OK.

    The Data Explorer dialog box appears.

    Note:    You can also create reports based on SQL/ODBC data sources, dictionary files, or query files. See The Crystal SQL Designer, and Dictionaries, for more information.
