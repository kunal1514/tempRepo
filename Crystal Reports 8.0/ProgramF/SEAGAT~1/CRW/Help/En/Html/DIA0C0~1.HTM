Link Options dialog box

This dialog box appears when you have a link line selected and you click the Options button in the Visual Linking Expert.

Use the Link Options dialog box to specify join types and the criteria the program is to use when linking a single table to two or more other tables (A to B, A to C).

Link description

This box describes the selected link. The field(s) listed on the left are the link from field(s) and the field(s) listed on the right are the link to field(s).

Index In Use

This list displays the index that is currently in use. If you click the arrow it also lists the other indexes that are available for the link as well as the option <no specific index>.

Fields In Index

This box lists the fields that are included in the index that is currently selected in the Index in Use list.

Allow partial text matches

This check box is for Paradox and dBASE users. Use it to specify whether the program should have partial text matching enabled when linking.

When linking to two files from this file options

Three different options for looking up records are available:

  • Look up both files at the same time
  • Look up all of one file, then all of the other
  • Look up all combinations of the two files.
SQL Join Type options

The program enables you to specify the type of join you want it to use when joining SQL tables.
