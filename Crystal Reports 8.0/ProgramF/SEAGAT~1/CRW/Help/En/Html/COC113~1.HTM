Stored Procedure Parameters command

Use the Stored Procedure Parameters command to change the parameters for the stored procedure you are using in your report.

You can build reports using stored procedures from the SQL data sources that support them. Seagate Crystal Reports assumes that the stored procedure you are using for creating a report will generate a result set, and it makes that result set available as a table. If the stored procedure requires parameters, those parameters are stored with the report.
