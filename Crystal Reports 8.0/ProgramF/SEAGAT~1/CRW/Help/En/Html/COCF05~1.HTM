Using the Summary Field command

  1. While in the Design tab, select the summary of interest.
  2. Choose the Summary Field command from the Edit menu. The Change Summary Operation dialog box appears.
  3. Select the desired summary operation. When you click OK, the program changes the summary to conform to your new selection.
