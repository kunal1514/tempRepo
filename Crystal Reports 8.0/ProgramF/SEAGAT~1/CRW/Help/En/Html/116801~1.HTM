Resizing objects using guidelines

If you snap an object to guidelines on two sides (top and bottom, or left and right), you can resize the object by dragging one or both of the guidelines. This is especially useful if you need to resize a number of objects similarly at the same time.

To resize objects using guidelines
  1. Snap the objects to the guidelines.
  2. Drag one of the guidelines to resize the objects.
