New Report tab

Use the New Report tab to set the default report directory and mail destination.

Report Directory

Use this box to set the default directory that the program uses when searching for and displaying existing reports.

Browse

Click this button to choose a report directory.

Mail Destination

This box lists all the possible destinations you can choose to export a report to e-mail. Click the arrow on the list and select a destination. Your choices are:

  • Application
  • Exchange Folder
  • Lotus Notes Database
  • Microsoft Mail (MAPI)
  • other mail destinations (depending on how your computer is configured).

When you have made a selection and open a mail-related dialog box in the program, it will open with your specified destination already selected.

Re-import Subreports When Opening Reports

When you select this check box, subreports are updated each time you open their containing reports. This option is not selected by default.
