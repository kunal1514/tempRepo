Using the Database Field command

  1. Choose the Database Field command from the Insert menu. The Field Explorer appears, with Database Fields highlighted, listing all of the fields in the active database.

    Note:    If you are working with Btrieve and using a .DDF file that contains more than four database files, only the file names, not the field names are displayed.

  2. Select the field of interest from the list.
  3. Click the Insert button. An object frame appears.
  4. Move the frame to the desired insertion point and click to place it.
  5. Repeat Steps 2-4 until you have inserted all the fields you want to insert.
  6. Click Close when finished.

Note:    For your convenience in setting up a report, the object frame is sized to approximate the true field width of the field selected. If the automatic field heading is longer than the field name (for example, the heading State over a two character state abbreviation), the frame is sized to match the width of the heading, not the width of the field itself.
