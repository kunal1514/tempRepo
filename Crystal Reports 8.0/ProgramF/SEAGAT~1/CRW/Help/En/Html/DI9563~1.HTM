Convert Database Driver dialog box

This dialog box appears when you choose the Convert Database Driver command from the Database menu.

Use the Convert Database Driver command to change the database driver with which you connect to a particular database, for example when moving from an ODBC driver to a native driver.

Convert Database Driver on next Refresh

Select this check box to change the driver the program uses when running your report. You cannot set the name of the driver unless you have selected this check box. This option lets you direct the program to use a different database driver (PD*.DLL):

  • the next time it refreshes data
  • if you are not saving data with the report, the next time you open the report.
To

Use this list to select the new driver.

OK button

When you click OK, the database driver is converted. Depending on the type of database driver you selected, you may be prompted for server or data source information.

The Map Fields dialog box appears if the report database has changed since you last ran the report.
