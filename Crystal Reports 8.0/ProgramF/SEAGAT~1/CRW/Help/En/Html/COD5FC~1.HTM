Change Map Layers command

Use the Change Layers command to open the Layer Control dialog box. Use this dialog box to rearrange the layers of your map. For example, you can rearrange the layers so that the Major Cities layer appears above the Highway layer, or so that the Capitals layer appears above the Top 20 Cities layer.

Note:    This command is available only when from the Analyzer Tab.
