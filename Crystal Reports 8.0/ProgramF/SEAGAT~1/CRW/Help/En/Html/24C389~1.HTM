Exchange Data Access

Seagate Crystal Reports has the ability to access various types of Microsoft Exchange data. The program is capable of retrieving valuable information from the Exchange Information Store, Message Tracking Logs, and Exchange Server Administrator Objects.

The report designer uses the Physical Database server type method to access exchange data. Thus, it will use the SQL/ODBC method to log on to the desired exchange server type and display those types in the Log On Server dialog box.

You can active this option by choosing:

  • Log On/Off Server from the File menu (when there is no report opened)
  • Log On/Off Server from the Database menu (when there is a report opened).

The following is a summary of the various types of Exchange data that the Report Designer can access.

  • Address Book Contents
  • Exchange Folder Contents
  • Message Tracking Logs
  • Exchange Administrator Data Server Types.
