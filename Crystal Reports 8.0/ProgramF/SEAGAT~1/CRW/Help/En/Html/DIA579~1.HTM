Insert Object dialog box (Create from File)

This dialog box appears when you choose the Create from File option in the Insert Object dialog box.

Use the Insert Object dialog box (Create from File) to insert an already existing object.

File

Use this box to type in the path and file name of the file you want to use as an OLE object. Be sure that the file you select can be used as an OLE object. If you are not sure, check the documentation for the application that was used to create the file to see if the application is an object application.

Browse

If you are not sure of the path and file name of the object file, click the Browse button. The Browse dialog box appears. Use the Drives, Directories, and File lists to locate the file you want to use. When you click OK, the program automatically writes the path and file name into the File box.

Link

Select the Link check box if you want to create a linked object. When a linked object is activated from within a report, the original file is opened inside the application that was used to create it. Any changes you make directly affect the original file also.

If you want the data in an object to remain available to other applications, and always reflect the most current changes to the data, remember to link the object to the report.

Display As Icon

Specifies whether the embedded object will appear in the report as an icon or whether the actual contents of the object will always be visible. Note that some object applications only display objects as icons.

If you select Display as Icon, you can open the object in your document by double-clicking its icon.

Change Icon

Click this button to change the icon that represents the embedded object in your report.
