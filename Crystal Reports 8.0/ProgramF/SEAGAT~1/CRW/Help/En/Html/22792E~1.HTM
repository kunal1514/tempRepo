To incorporate a parameter into a formula

  1. Create the parameter.
  2. Check to make sure you are working in the Design tab.
  3. On the Insert menu, click Formula Field.

    The Field Explorer appears.

  4. Click New.

    The Formula Name dialog box appears.

  5. Enter the name of the formula, then click OK.
  1. Create a formula using the parameter field as you would any constant value. For example, rather than creating a formula that hard-codes the country name:

    {customer.COUNTRY} = "USA"

    Use a parameter field instead of "USA".

    {customer.COUNTRY} = {?Country}

    To do this double-click the database field, press =, then double-click the parameter.

  2. Click Save, then Close.

    The Field Explorer dialog box appears. The name of the formula you have just created is highlighted in the Formula list box.

  3. Drag and drop the formula into the report.
  4. Click Close to exit the Field explorer dialog box.
  5. Click Refresh to generate the report.

    A dialog box appears, prompting you for values.
