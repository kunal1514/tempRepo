Lotus Domino

A Lotus Domino database can be read by Seagate Crystal Reports through ODBC. For Windows 95, Windows 98, and Windows NT, the Lotus Domino DBMS translation layer consists of three files:

  • Nsql32.dll
  • Nsqlv32.dll
  • Nsqlc32.dll

These files use the drivers installed by the Lotus Notes client to work with the Lotus Domino database. The client component of Lotus Domino must be installed on the local machine.

Note:    The Domino Server can be substituted for the Lotus Notes client in the database layer of this diagram.
