Change Map Title dialog box

The Change Map Title dialog box opens when you choose the Change Map Title command from the Analyzer menu, or from the map's right-click menu. Change Map Title is available only when you are working with a map on the Analyzer tab.

Use this dialog box to specify or change the title you'd like to use for your map. The dialog box is empty if you did not specify a map title on the Text tab of the Map Expert. Otherwise, the title you specified earlier appears in the Change Map Title dialog box.

Enter a new title, or edit an existing one in the dialog box and click OK when you are done. The new title appears on your map in the Analyzer and Preview tabs.
