Analyzer Tools toolbar

The Analyzer Tools toolbar allows you to quickly view and customize maps and charts. To display the Analyzer Tools toolbar, choose the Toolbars command from the View menu and then toggle the Analyzer check box on in the Toolbars dialog box.

Some of these buttons operate on charts, some on maps. Some operate on maps or charts in any tab, others only operate in the Analyzer tab. To display the Analyzer Tab click on the map or chart you want to work with, and choose Launch Analyzer from the Analyzer menu.

One drop-down list appears at the left-hand side of the Analyzer Tools toolbar.

Use this drop-down list to change the color of a single item on a chart in the Analyzer tab (for example, to change the color of one of the bars on a bar chart).

The buttons on the Analyzer Tools toolbar perform the following functions:

Opens the Choose a Chart type dialog box for changing the type of chart displayed.

Opens the Chart Options dialog box, for setting the general properties of the chart.

Opens the Titles dialog box for adding titles to the chart and its axes.

Opens the Series Options dialog box for setting properties for the selected data series on a chart in the Analyzer tab.

Opens the Numeric Axes, Grids & Scales dialog box for setting these properties of the chart.

Opens the Formatting dialog box for setting font line and fill properties of the selected item on a chart in the Analyzer tab.

Opens the Choose a Viewing Angle dialog box for changing the viewing angle of a surface chart.

Activates the drill-down mode, and changes the cursor to a Drill-down cursor. Shortcut to the Select Mode command.

Activates the zoom in mode, and changes the cursor to a Zoom cursor. Shortcut to the Zoom In command.

Activates the zoom out mode, and changes the cursor to a Zoom cursor. Shortcut to the Zoom Out command.

Activates the Pan mode, and changes the cursor to a Panning cursor. Shortcut to the Pan command.

Centers the map. Shortcut to the Center Map command.

Opens the Customize Map dialog box, for changing the style and appearance of the map on the Analyzer Tab. Shortcut to the Change Map Type command.

Opens the Layer Control dialog box, for rearranging the layers of the map on the Analyzer Tab. Shortcut to the Change Map Layers command.

Opens the Map Region Resolver for resolving mismatches between the names of regions in the data and those on the map. Shortcut to the Resolve Mismatch command.
