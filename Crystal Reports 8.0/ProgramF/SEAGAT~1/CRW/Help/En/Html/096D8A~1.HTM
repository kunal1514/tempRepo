To create a running total for a group

  1. To get started, create a report using the sample data, Xtreme.mdb located in the File Data Sources folder of the Data Explorer dialog box.

    Link the Customers and Orders tables and place the following fields from left to right in the Details section:

{customer.CUSTOMER NAME}
{orders.ORDER ID}
{orders.ORDER AMOUNT}
  1. On the Insert menu, click Group and group on the {customer.CUSTOMER NAME} field.
  2. On the Insert menu, click Running Total Field.

    The Field Explorer dialog box appears.

  3. Double-click Running Total Fields.

    The Create Running Total Field dialog box appears.

  4. Enter the name "GroupRunningTotal" in the Running Total Name box.
  5. Highlight {orders.ORDER AMOUNT} in the Available Tables and Fields box, and use the first arrow button to move it over to the Field to summarize box.
  6. Select sum from the Type of summary list.
  7. In the Evaluate section of the dialog box, click For each record.
  8. In the Reset section of the dialog box, click On change of group and accept the default group name.
  9. Click OK to save the running total field.

    You return to the Field Explorer dialog box.

  10. Place the running total field in the Details section of your report, just to the right of the {orders.ORDER AMOUNT} field.

    Note:    If you want to view a grand total of each group, place the running total field in the Group Footer section of your report.
