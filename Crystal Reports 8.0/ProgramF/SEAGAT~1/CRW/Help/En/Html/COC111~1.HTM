Curve Fit & Stat Lines command

On the Analyzer tab, use the Curve Fit & Stat Lines command to fit one or more statistical lines or curves to a series of data on the chart.

Note:    This command is only available when you select a data series on a chart, by clicking one of the data items for the series.

Choosing the Curve Fit & Stat Lines command opens the Curve Fit & Stat Lines dialog box for you to select the type of curve to display for the chosen data series.

You can set the color of the curves for a data series to the color used for that data series on the chart. (If you select this option for a chart that colors each item differently, such as a bar chart, the color for the first data item is used.)

If you do not select Show Color Same as Series for a data series, you can set the color for its curves selecting one of them and using the Change Color list.
