Edit Parameter Field dialog box

This dialog box appears when you select a parameter from the Parameter Fields list in the Field Explorer and then click the Edit button. It also appears when you select a parameter in a report and choose Parameter from the Edit menu, or when you right-click a parameter and choose Edit Parameter from the shortcut menu.

Use the Edit Parameter Field dialog box to change the prompting text, value type, default value(s), and/or option for the selected parameter field.

Name

When editing a parameter field, this box displays the parameter's name, but is not available for you to change the name.

Prompting text

The current prompting text for the selected parameter field is shown in this box. This is the message that appears when the program prompts you for a value. Edit the prompting text to fit your needs.

Allow multiple values

By selecting the Allow multiple values check box, you can select more than one default value for your parameter field after you click the Refresh button on the Standard toolbar. This option is not selected by default.

Discrete value(s)

By selecting the Discrete value(s) option, you can select only singular values for your parameter field. If you have Allow multiple values selected, you will be able to select more than one discrete value for your parameter field, though those discrete values will continue to function singularly as opposed to a range value. You can choose discrete values for your parameter field after you click the Refresh button on the Standard toolbar. This option is selected by default.

Range value(s)

By selecting the Range value(s) option, you can select a range of values for your parameter field. For example, if you've selected a currency field, your parameter field (with range value selected) could equal any number that falls between a lower and upper limit you set.

Discrete and Range Values

By selecting the Discrete and Range Values option, you can enter both discrete values and values in ranges.

Allow editing of default values when there is more than one value

With the Allow editing of default values when there is more than one value check box selected, you can edit values in the parameter field after the parameter field has been added to the report. To edit default values, click the Refresh button and select Prompt for new parameter values. This check box is selected by default.

Set Default Values

Click this button to open the Set Default Values dialog box. Use the Set Default Values dialog box to create a list of default values for the parameter field you are creating.
