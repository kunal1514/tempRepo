To update a cube location in an OLAP grid object

  1. Select the grid by clicking on the border.
  1. From the Database menu, click Set OLAP Cube Location.

    The Confirm Command dialog box appears.

  2. Click Yes.

    The Set OLAP Database Location dialog box appears.

  3. Click Select.

    The Choose OLAP Type dialog box appears.

  4. Select the OLAP Type.
  5. Click OK.

    The Connect To Server dialog box appears.

  6. Enter the OLAP Server, User ID, and Password.

    Note:    The fields will vary depending on the OLAP type you are using.

  7. Click Connect.

    The Chose Cube dialog box appears.

  8. Select a cube.
  9. Click OK.

    The Set OLAP Location dialog box appears.

  10. Click OK.

    The Propagate Set Location dialog box appears.

  11. Click Yes to update the remaining OLAP grids and follow the updating the database location steps; otherwise click No to return to the report.
Updating the database location
  1. Click Yes to update the database location.

    The Set Location dialog box appears.

  2. Select the database, then click Set Location.

    The Data Explorer appears.

  3. Expand the directory for the cube.
  4. Select the table, then click Set.

    The Connect to Server dialog box appears.

  5. Enter the OLAP Server, User ID, and Password.

Note:    The fields will vary depending on the OLAP type you are using.

  1. Click Connect.

    The Chose Cube Dialog box appears.

  2. Select the cube, then click OK.
  3. The Propagate Set Location dialog box appears with a message asking you to propagate the server and database changes across tables with the same original information.
  4. Click Yes.
