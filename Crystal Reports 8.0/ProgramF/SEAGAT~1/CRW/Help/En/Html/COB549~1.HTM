Group Name Field command

Use the Group Name Field command to create live group headers. Once you have selected a group, the program creates a group name object for that group. The object contains the field or formula value that triggers each grouping (that is, the value in the sort and group by field). If you place this object in the group header section for the group, the program will print a live group header at the beginning of each group.

A live group header is a header that changes dynamically when the content of the group changes. If you were to group your data by state, for example, a typical live group header would print the name of the state at the beginning of each group.

By default, the program automatically inserts a group name field in the group header section whenever you create a group.

When you choose the Group Name Field command, the Field Explorer appears with Group Name Fields selected.

Note:    You can also open the Field Explorer by clicking the Insert Fields button on the Standard toolbar.
