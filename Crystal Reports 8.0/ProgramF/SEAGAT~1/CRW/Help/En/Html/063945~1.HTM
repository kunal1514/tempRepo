Special fields

To display information such as Page Numbers, Print Date, and Report Comments use the commands in the Special Fields tree view of the Field Explorer dialog box.

To insert a special field

Seagate Crystal Reports allows you to easily insert Page Number, Record Number, Group Number, Print Date, and Total Page Count fields, among others, into your report.

  1. On the Standard toolbar, click Insert Fields.

    The Field Explorer dialog box appears with Database Fields selected.

  2. Scroll down to Special Fields and expand it by clicking.

  3. Choose a command from the Special Field list to insert in the report.

    Each special field is inserted into the report as an object. An object frame appears. You can now place it on the report.

Note:     To change the formatting of an inserted object, click the object to select it and click Object Properties on the Supplementary toolbar. The Format Editor appears where you can make the desired changes. See Formatting.
