Select a Field for Dimension dialog box

This dialog box appears when you click the Select button on the Other Dim Tab of the Insert OLAP Grid Object dialog box.

Use the Select a Field for Dimension dialog box to select a field value for the OLAP Grid other dimension that you have selected in the Insert OLAP Grid Object dialog box.

Field Values

Use this list to select the desired field value for the other dimension.
