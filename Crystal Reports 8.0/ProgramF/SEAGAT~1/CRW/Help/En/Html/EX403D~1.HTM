Text tab

The Text tab allows you to determine the title of your map, legend information, and the legend title.

Map title

Use this text box to create a title for your map. Creating a map title is optional.

Full Legend

Select the Full Legend option button for a detailed breakdown of how the data has been represented in the map.

Compact Legend

Select the Compact Legend option button for a minimal breakdown of how the data has been represented in the map.

No Legend

Select this option button if you prefer not to have a legend with your map.

Made by map

Select the Made by map option button to create a legend title for you.

Specify

Select the Specify option button to create your own legend title.

Title

Use this text box to create a legend title for your map.

Subtitle

Use this text box to create a legend subtitle for your map
