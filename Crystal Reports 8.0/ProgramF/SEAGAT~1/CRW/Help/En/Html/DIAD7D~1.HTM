Tabs tab (Format Editor)

Use the Tabs tab to set, delete, modify and edit tabs in a text object.

If your text object is in edit mode, this tab appears on the Text Format dialog box.

Tab Stop Position box

Enter the desired tab stop position in this box. Tab stops are measured in inches.

Alignment options

Choose from four alignment option for the selected tab stop: left, right, center, and decimal.

Set button

Once you have entered a tab stop position into the Tab Stop Position box, click this button to set it.

Delete button

When you click this button, the selected tab stop is deleted.

Delete All button

When you click this button all the defined tab stops are deleted.
