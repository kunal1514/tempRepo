Address Book contents

Description

Accesses account's personal information for all entries in the address book. All of the address books are on the Exchange Server, except for the Personal Address Book. Each of the address books are "Recipients Containers" on the Server and their contents can be viewed by selecting them in the Exchange Administrator.

The report designer interprets the information in the following manner:

  • the selected address book is seen as a database table
  • each object contained within the address book is seen as a record in the address book table
  • each object's properties are seen as fields of the address book table's records.

To view the contents of an address book, select the Recipients Container that represents the desired address book on the Exchange Server.
