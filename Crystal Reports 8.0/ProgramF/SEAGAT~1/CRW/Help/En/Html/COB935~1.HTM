Undo command

The Undo command is a dynamic command. The command name (as seen on the menu) changes based on the last action you performed. Undo reverses the effects of an action and restores your document to the way it appeared before you performed the action. The program supports multiple levels of Undo.

Note:    This command is also available as a button on the Standard toolbar.
