Object Size and Position dialog box

Use the Object Size and Position dialog box to change the height and width of the selected object, and to adjust the object's position from the top and side margins. All units are in inches.

X

Use the X box to change an object's horizontal position from the margin. All units are in inches.

Y

Use the Y box to change an object's vertical position from the margin. All units are in inches.

Height

Use the Height box to change an object's vertical height. All units are inches.

Width

Use the Width box to change an object's horizontal width. All units are inches.
