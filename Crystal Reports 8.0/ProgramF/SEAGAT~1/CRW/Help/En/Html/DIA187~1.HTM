Group Selection Formula Editor

This dialog box appears whenever you choose the Group command from the Edit Selection Formula menu (under the Report menu).

Use the Group Selection Formula Editor to edit a group selection formula that limits the groups the program uses when preparing your report.

You can use the full range of report fields, operators, and functions to create a formula that restricts the range of data to print.

Note:    

  • You can only use Crystal syntax when creating a formula.
  • No field placement box appears when you create a selection formula; the formula is simply stored in the report.
  • Your group selection formula must be Boolean, that is, it must result in a Yes or No answer.
  • You cannot use commas in numbers you enter in a formula. Enter only the number itself.
