Magnification Factor dialog box

This dialog box appears when you choose the Zoom command from the View menu.

Use the Magnification Factor dialog box to set the magnification factor for zooming in or out of your report.

Magnification Factor

Click this option to set the magnification to a particular percentage. Type the magnification factor into the box, for example 50%.

Fit One Dimension

Click this option to fit the page according to the width of the report in the Design or Preview tabs.

Fit Whole Page

Click this option to fit the whole page of the report in the Design or Preview tabs.

Reset

Click this button to reset the magnification factor to the default value of 100%.
