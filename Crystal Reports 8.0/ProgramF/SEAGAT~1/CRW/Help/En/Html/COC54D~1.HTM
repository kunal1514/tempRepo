Picture command

Use the Picture command to insert bitmapped artwork (graphics, logos, and so on) into your report. The program helps you enhance the visual impact of your report using artwork in the following popular formats.

When you use the OLE Object command on the Insert menu to place a picture in a report, the picture can be treated as either an embedded or a linked object.

When you use the Picture command on the Insert menu to insert a picture into the report, the Report Designer converts the picture into a static object.

Static objects do not support editing. To edit a static object, it has to be converted into an editable type of object using the Convert Picture Object command on the Edit menu.

Note:    Seagate Crystal Reports supports Bitmap, TIFF, JPEG, and PNG files.
