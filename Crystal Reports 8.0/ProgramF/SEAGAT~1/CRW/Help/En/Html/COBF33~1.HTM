Links command

Use the Links command to make changes in existing OLE object links. Using this command, you can change the source data for a link and also the way in which that link is updated. This command is only active when you have a linked object in the report. When you insert an OLE object, you can choose to insert it as an embedded object or as linked object. See Insert Object dialog box (Create from File).

When you choose the Links command, the Links dialog box appears.

Note:    This command is not related to editing a database link. To edit a database link you must choose the Visual Linking Expert command from the Database menu.
