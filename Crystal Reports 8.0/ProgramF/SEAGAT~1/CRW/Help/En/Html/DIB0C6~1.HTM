Number and Date Format dialog box

This dialog box appears after you click OK in the Character Separated Values dialog box when exporting a report.

Use the Number and Date Format dialog box to:

  • set the number formats you have defined in your report to be used to export your data to the new disk file
  • set the date formats you have defined in your report to be used to export your data to the new disk file.
Same number format as in report

Select this check box if you want the number formats you have defined in your report to be used when exporting your data.

Same date format as in report

Select this check box if you want the date formats you have defined in your report to be used when exporting your data.
