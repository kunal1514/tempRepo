About the Microsoft Excel Add-In

Choose an entire data range in a spreadsheet, or select certain cells to create a Seagate Crystal Report. The report you create is linked to your Excel spreadsheet and can be refreshed to show changes made to the spreadsheet data.

Note:    Excel Add-In reports are updated when you refresh their data in Seagate Crystal Reports if you have:

  • added a row to the selected spreadsheet range (but not pre-appended or appended a row)
  • deleted a row from the selected spreadsheet range
  • modified a cell within the selected spreadsheet range.

After you create a report using the Wizard, you can preview it using the default ActiveX viewer or you can launch Seagate Crystal Reports or Seagate Analysis to modify it.
