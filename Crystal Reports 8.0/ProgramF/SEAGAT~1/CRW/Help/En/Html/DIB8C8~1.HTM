Symbol Style dialog box

The Symbol Style dialog box appears when you click Graduated on the Type tab of the Map Expert and then click Customize. The Symbol Style dialog box offers a variety of formatting and style options for your Graduated map symbol, including:

  • font type
  • font size
  • symbol options
  • color
  • rotation angle
  • background options
  • effects options.
Font

Use this list to change the font of your Graduated map symbol. The default font is Arial.

Font size

Use this list to change the font size of your Graduated map symbol. The default font size is 18.

Symbol

Use this list to change the symbol for your Graduated map. You have 224 symbols to choose from.

Color

Use this list to change the color of the symbol for your Graduated map.

Sample

The sample to the right of Font, Symbol, and Color shows you how the symbol for your Graduated map is affected by changes you make in the Symbol Style dialog box.

Rotation Angle

Use this box to change the rotation angle of your symbol as it appears on the Graduated map.

None

Use this option to display your Graduated map symbol without a background.

Halo

Use this option to display your Graduated map symbol with a Halo background.

Note:    Halo backgrounds are most effective when the background color greatly contrasts the symbol color. For example, a white or yellow halo background will give a true halo effect to a navy blue symbol.

Box

Use this option to display your Graduated map symbol in a colored-background box.

Color

Use this list to select the background color for your Graduated map symbol.

Effects options

Use these check boxes to apply the following effects to your Graduated map symbol:

  • Bold
  • Italic
  • Drop Shadow.
