Group option view

The Group option view allows you to show a summary field (such as Last Year's Sales) on change of a geographic field (such as Region). In order to create a map using the Group layout, you must have at least one group and at least one summary field in your report.

On change of

Use the on change of list to specify a field to use as a condition for displaying a summary field. Summary values will be displayed whenever the value in the condition field changes.

Show

The Show list displays the value options for your map. In other words, the data contained in the field in the Show list will be used by the program as your map values. If the Show list contains the Sum of Customer.Last Year's Sales field, your map values will be last year's sales for each customer.
