Change Summary Operation dialog box

This dialog box appears when you use the Change Summary Operation command. The command is available when you:

  • right-click a summary field and choose the Change Summary Operation command from the shortcut menu,
  • when you choose Summary Field command from the Edit menu,
  • when you click the Set Summary Operation button on the Data tab of the Map Expert. This button is only active when you have cleared the Don't summarize values check box, and when you have selected a field in the Map Values list.

Use the Change Summary Operation dialog box to set or change the summary operation for the selected field.

Summary

Select a summary operation from the list. Use the Summary list to set the summary operation for the summary field.
