Informix Online Server

The Informix Online Server provides client/server capabilities along with SQL compliance.

Seagate Crystal Reports opens SQL data created by Informix through the P2sifmx.dll, installed with Seagate Crystal Reports. This translation file works with the Informix database drivers to read Informix data. You must have the Informix client installed and configured properly before using this translation file.
