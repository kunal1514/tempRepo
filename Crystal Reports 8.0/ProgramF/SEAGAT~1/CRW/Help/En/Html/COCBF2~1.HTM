Using the Log On/Off Server command

To log off a server
  1. Choose the Log On/Off Server command from the Database menu.
  2. The Data Explorer shows the various server types and the servers that are available to your system. Browse through the folders to the server you wish to log off.
  3. Click Log off.
To log on to a server
  1. Choose the Log On/Off Server command from the Database menu.
  2. The Data Explorer shows the various server types and the servers that are available to your system. Browse through the folders to the server you wish to log off.
  3. Click Log on.
  4. A dialog box appears, requesting server-specific login information. Use this dialog box to identify yourself and to specify the database you want to activate.
  5. Enter the requested login information and click OK. The program logs you onto the specified server and returns you to the Design tab.

Note:    If you want to activate multiple databases from the same server, you need to log on to the server each time you want to activate another database.
