Grid options

Snap To Grid

When you select this check box, new database fields you insert snap to the nearest grid positions. When you clear this check box, new database fields are left where you place them. This option is not selected by default.

Show Grid in Design

When you select this check box, a grid is displayed on the Design tab. This option is not selected by default.

Show Grid in Preview

When you select this check box, a grid is displayed on the Preview tab. This option is not selected by default.

Free-Form Placement

When you select this check box, objects in your report can be placed anywhere in their respective sections. This option is selected by default.

Grid Size

This box lets you specify the size of the grid increments. The unit of measure used for grid increments is based on your settings in the International section of the Windows Control Panel.
