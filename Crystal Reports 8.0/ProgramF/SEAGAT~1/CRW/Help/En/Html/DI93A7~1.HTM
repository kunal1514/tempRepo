Change Icon dialog box

This dialog box appears when you click the Change Icon button in the Convert dialog box.

Use the Change Icon dialog box to change the appearance of the icon you are using to represent an OLE object in your report.

Current

Displays the current icon in your report.

Default

Displays the default icon in your report.

From File

Enables you to select an icon from the specified file. You can choose any icon in an icon file, a library file, or an executable file. Select the icon from the scroll box below the file name. Click the Browse button to select a different file.

Label

Specifies the icon label that appears in the report.

Browse

Click the Browse button to select a different file containing icons. You can use choose one of the icons in an icon file, a library file, or an executable file.
