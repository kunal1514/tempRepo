OLAP option view

On change of

Select the field you want to use as a condition for plotting values in a graph or chart. Points will be plotted whenever the value in the condition field changes. For example, if you create a graph of last year's sales for all your customers, you will select the Customer Name field as the condition and the Last Year's Sales field as the value. Whenever the Customer Name changes a point will be plotted according to the value of their last year's sales.

Subdivided by

Use this list to choose the secondary row or column to base your chart on. Clicking this box displays a list of the row/columns fields in your report that you can choose from.

Other Dimensions

Click the Other Dimensions button to call the Format Other Dimensions dialog box.

Click a dimension (shown with its field setting) in the list then click Select a field value to specify a new field for the dimension to be fixed to.
