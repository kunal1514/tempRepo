Using the Save Data with Report command

Use the Save Data with Report command to save the report definition and the underlying data with your report. When this command is active, then when the report is opened again, it will include the data that was available the last time it was saved.

When this command is inactive, the data will refresh each time the report is opened. The report will include the most recent data available.

  1. Choose the Save Data with Report command command from the File menu.

    A check shows that the Save Data with Report option is active. By default, this option is active.

  2. Switch the check mark on/off as desired. The program will save data accordingly.
  • If you save your data with the report, your report will require more disk space (enough for the report template and the data), but the program will not need to retrieve the data before printing.
  • If you save only the definition, your report will require somewhat less disk space (enough for the template only) but the program will need to retrieve the data before it prints the report.

Note:    When the program saves the data with the report, it compresses the data in order to take less disk space. It decompresses the data when it opens the report.
