Align submenu

Use the commands on the Align submenu to align the currently selected objects with the last object you select. Click an object to select it and then Shift-click or Ctrl-click to select further objects.

Once you have selected the objects that you want to align, choose a command from the submenu.

Tops

Use the (Align) Tops command to align the tops of the selected objects with the top of the last object selected.

Middles

Use the (Align) Middles command to align the (vertical) middles of the selected objects with the (vertical) middle of the last object selected.

Bottoms

Use the (Align) Bottoms command to align the bottoms of the selected objects with the bottom of the last object selected.

Baseline

Use the (Align) Baseline command to align the baseline of the selected objects with the baseline of the last object selected.

Lefts

Use the (Align) Lefts command to align the left of the selected objects with the left of the last object selected.

Centers

Use the (Align) Centers command to align the center of the selected objects with the center of the last object selected.

Rights

Use the (Align) Rights command to align the right of the selected objects with the right of the last object selected.

To Grid

Use the (Align) To Grid command to align the selected objects with the grid.
