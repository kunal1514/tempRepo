Data tab

The Data tab appears in the Report Expert when you click any of the Expert buttons in the Seagate Crystal Report Gallery when creating a new report.

Use the Data tab to select the data source and tables (databases in Essbase terminology) and add indexes to tables in your new report.

Data Sources

The available data sources are shown as buttons. To select a data source, click the corresponding button.

Tables

Once you have chosen the desired data source, the tables (databases in Essbase terminology) will be listed here.

Remove

Click this button to delete a table (database in Essbase terminology) selected in the Tables list.

Add Index

Click this button to add an index to a PC-type database (datafile). Indexes are used for linking.
