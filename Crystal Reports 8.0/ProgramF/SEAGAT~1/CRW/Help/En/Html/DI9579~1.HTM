Choose Field dialog box

This dialog box appears when you choose the Select Expert command on the Report menu if you do not have a field selected.

Use the Choose Field dialog box to select a field for record selection.

Fields

This list displays the report fields available in your report for use in record selection.

Browse

Click Browse to browse the data in the selected field.
