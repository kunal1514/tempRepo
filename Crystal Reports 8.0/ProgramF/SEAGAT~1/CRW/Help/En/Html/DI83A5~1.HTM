Allow Reporting On dialog box

This dialog box only appears when you have selected the Reprompt user when connecting check box on the Database tab of the Options dialog box. When you log on to a server and add a table by using the Add Database to Report command from the Database menu, the Allow Reporting On dialog box appears.

Use the options in this dialog box to select the data that you want to report on.

Tables

When you select this check box, the program will report on database tables. This option is selected by default.

Views

When you select this check box, the program will report on virtual tables. This option is selected by default.

Synonyms

When you select this check box, the program will allow you to report on the virtual tables that are available on some hosts. This option is not selected by default.

System Tables

When you select this check box, the program will report on system tables. These tables are typically used by the system administrator only, but are available for use if you have the appropriate permissions. This option is not selected by default.

Procedures

When you select this check box, the program will allow you to report on the result sets from stored procedures (if you are using SQL systems that support stored procedures).

Table Name LIKE

This box allows you to enter the SQL LIKE function to specify the kinds of table names you want to appear in the Data Explorer. You can use the underscore character (_) or the percent sign character (%) as wildcards with this function. The underscore character specifies any single character while the percent sign signifies any character string. For example, DAV_ would match DAVE but not DAVI; whereas DAV% would match both DAVE and DAVID. A table name Like C% would display only those tables that have a table name beginning with the letter C.

Creator LIKE

This box works exactly like Table name Like, except that the LIKE function is used to select the Creator (or Alias) of the table, not the table name itself. For example, Creator Like C% would display only those tables that had an owner beginning with the letter C.

COUNT the tables first

This option will count the tables and give you a warning if there are more than the specified number.
