Style tab (OLAP and Cross-Tab)

Use the Style tab to choose a style for your report.

This tab appears on the Cross-Tab Report Expert and on the OLAP Report Expert. It also appears on the Format OLAP Grid Object dialog box and the Format Cross-Tab dialog box.

Style

This box displays the formatting styles available for use in your report. Select the desired style from the list.

Sample Preview

This box displays a sample of the style selected in the Style list.
