Using the Page Setup command

  1. Choose the Page Setup command from the File menu. The Page Setup dialog box appears with the current margin settings displayed and the Use Default Margins check box selected.
  2. Accept the default margins or enter new margins.
  • To accept the default margins, click OK. The program returns you to the Design tab.
  • To change the margins, enter your new margins in the Top, Left, Bottom, and/or Right Margin edit boxes. When you change any of the default margins, the program clears the Use Default Margins check box.
  • To restore the margins to their default settings, select the Use Default Margins check box.
  1. Click OK. The program returns you to the Design tab where your new margins are now in place.
