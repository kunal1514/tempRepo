Distributing Reports

Seagate Crystal Reports enables you to distribute your report using a variety of methods. This section covers how to:

  • print a report
  • fax a report
  • export a report
  • save a report in Seagate Crystal Reports 7 format
  • open and save reports using a web folder
  • use a web browser with a Report Viewer to view a report.
