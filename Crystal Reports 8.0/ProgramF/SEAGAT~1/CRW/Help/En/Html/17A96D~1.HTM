Using the Report Viewer for ActiveX

  1. Type the URL for the report into your web browser. Obtain this from your Systems Administrator or Information Systems Manager.

    The first page of the requested report appears inside the Report Viewer. A Smart Navigation Group Tree is generated based on the groups in the report.

  2. Expand the tree and click a group to navigate to a specific part of the report.

Note:    You may not have access to Smart Navigation with the Group Tree. Smart Navigation may be disabled for the report or for the Report Viewer itself.

In addition to the functions provided on the Standard toolbar, this Report Viewer has the following extra functions in its customized toolbar:

  • Close current view
  • Print
  • Refresh
  • Show/hide group tree
  • Zoom control
  • Go to first page
  • Go to previous page
  • Specify page number
  • Go to next page
  • Go to last page
  • Stop
  • Search text.

Place the mouse pointer over any of the buttons on the customized toolbar to see the associated ToolTip.

Note:    If the report has been created using a different type of printer than the one available from your system, you may experience formatting problems when attempting to print a report displayed by the Report Viewer for ActiveX. The Report Viewer Print button uses printer settings saved with the report file. If you experience problems printing a report, contact your web server administrator.
