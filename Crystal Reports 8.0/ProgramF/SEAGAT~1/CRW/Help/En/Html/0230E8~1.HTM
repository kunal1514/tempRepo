Enhanced Formula Editor

These enhancements are available in all editions of Seagate Crystal Reports.

More flexibility

The formula language has been expanded to include case statements, looping and dynamic arrays.

Use the formula syntax you know best: Crystal or Basic

Crystal syntax is the formula language used in previous versions of Seagate Crystal Reports while Basic syntax is new to Seagate Crystal Reports 8. If you have worked with Microsoft Visual Basic, then Basic syntax may be more familiar to you.

You can create formulas using either Crystal or Basic syntax. Almost any formula written with one can be created with the other. As with Crystal syntax, Basic syntax does not require any additional DLLs to run.

New formula functions

The following groups of functions have more functions to choose from:

  • Financial functions
  • Math functions
  • Program shortcut functions
  • Type conversion functions
  • Date and Time functions
  • Array and String functions
  • Percentage functions.

New operators

The following groups of operators have more operators to choose from:

  • DateTime Literals
  • Array Operators
  • Operators for Unbound Ranges.

New control structures

Control structures similar to the Microsoft Visual Basic counterparts have been added to both Crystal and Basic syntax.

  • "For" statements
  • "Do" and "While" statements
  • "Select" statements.
