Working with Web Folders

You have the ability to open and save reports using Web Folders if you:

  • are running Windows 2000 or have Office 2000 installed
  • have access to a web server that is configured to support Web Folders
  • add a Web Folder from this server into your Web Folders folder.
To open your report
  1. On the File menu, click Open.

    The Open dialog box appears.

  2. Click Web Folders.
  3. Open the folder that contains the report.
  4. Double-click to open the report.
To save your report
  1. On the File menu, click Save As.

    The Save As dialog box appears.

  2. Click Web Folders.
  3. Locate the folder you would like to save the report to.
  4. Enter the file name.
  5. Click Save.

Note:    After making changes to a report in a Web Folder, you must save the changes to the same file in the same Web Folder.
