OLAP Report Expert

The OLAP Report Expert lets you create a report in which your OLAP data is displayed as a grid object. Although similar to the Cross-Tab Report Expert in several ways, the OLAP Report Expert appears to be different due to the requirements of working with OLAP data sources. You first specify the location of you OLAP data, and then you choose the dimensions you want to include in the grid. Next you filter the report data and choose and customize the style of the grid object. Finally, you can define labels for your grid and insert a chart, if you wish.

The Format OLAP Grid Object dialog box has the same functionality as the OLAP Report Expert and appears when you right-click a grid object then click Format OLAP Grid on the shortcut menu.

The Insert OLAP Grid Object Expert has the same functionality as the OLAP Report Expert and appears when you click the OLAP Grid command on the Insert menu.

The OLAP Report Expert has the following tabs:

  • Data tab (OLAP)
  • Rows/Columns tab (OLAP)
  • Filter/Page tab (OLAP)
  • Style tab (OLAP and Cross-Tab)
  • Customize Style tab (OLAP and Cross-Tab)
  • Labels tab (OLAP)
  • Chart tab
