Top N tab

Use the Top N tab to choose fields to sort and group information in your report by.

For this group

Select a sorting option:

  • Sort All groups
  • Sort the Top N groups
  • Sort the Bottom N groups
based on

Displays a list of summary fields in your report. Select the field you want to sort on from this list. When Sort All groups is selected, use the scroll box below the based on list to see the fields chosen for sorting.

where N is

Type the number of Top N or Bottom N groups to sort.

include other groups with the label

Select this option if you want to include all fields that do not fit into your Top N or Bottom N selection under a specified group name. Type the group name you want all fields that do not fit into your Top N or Bottom N selection to appear under.

Delete

Click this button to delete the selected summary field. This is only available when Sort All groups is selected.

Ascending/Descending

Select the Ascending option to sort the fields in ascending (A to Z, 1 to 9) order.

Select the Descending option to sort the fields in descending (Z to A, 9 to 1) order.
