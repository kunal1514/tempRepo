Others dialog box

This dialog box appears when you click the Options button on the Group tab of the Report Expert.

Use the Others dialog box to specify what to do with those records that do not fit into the specified group. You can discard the records, place them together in a newly named group, or leave them as is.

Discard all others

When you click this option, the program will not print any groups not specified in the sort order list. If you summarize groups, values in the discarded groups will not be summarized nor will they be included in any grand total if you have one.

Put all others together, with the name

When you click this option, the program will create a group that will lump together all groups not specified in the sort order list. This group will appear after the specified groups. If you summarize groups, the values in the group will be summarized and will be included in any grand total. You can specify the name of the group using the Name edit box.

Put all others together, with the name

Use this box to specify the group name for the Others group. By default, the group will be named Others. To specify a group name, select the name Others, and enter the desired group name in the edit box.

Leave in their own groups

When you click this option, the program will print your specified groups in the order you specified them, and it will then group and print the remaining groups normally. If you summarized groups, the values in the group will be summarized and will be included in any grand total.
