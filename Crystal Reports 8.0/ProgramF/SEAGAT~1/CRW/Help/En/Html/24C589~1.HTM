Exchange Administrator Data Server types

When you log on to any of the Exchange Administrator Data server types, you must select a profile (or profiles). Each profile that you select represents an Exchange Server. So, if you want to access n Exchange Servers, you would want to create n profiles where each profile's server value is one of the desired machines. Each one that you create for any of the Exchange Administrator Data server types will represent the profile (s)/Exchange Server(s) that you selected.

Seagate Crystal Reports can access a number of different Exchange Administrator Data Server types. These include:

  • Public Folder Allocation and Usage
  • Public Folder Membership
  • Public Folder Replication
  • Mailbox Properties.
