Others tab

Use the Others tab to set up what you want the program to do with all values not included in any of the specified groups.

Discard all others

Select this option if you do not want to print the unselected fields in the groups for your report. If you summarize groups, values in the discarded groups will not be summarized nor will they be included in any grand total if you have one.

Put them all together, with the name

Select this option and enter a name if you want to put all unselected fields in one group. The default name for this group is Others. If you summarize groups, the values in the group will be summarized and will be included in any grand total.

Leave in their own groups

When you click this option, the your specified groups are shown in the order you specified. Remaining groups are displayed normally. If you summarized groups, the values in the group will be summarized and will be included in any grand total.
