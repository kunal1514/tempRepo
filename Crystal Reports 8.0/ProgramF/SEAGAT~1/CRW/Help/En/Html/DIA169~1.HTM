Insert Grand Total dialog box

This dialog box appears when:

  • you choose the Grand Total command from the Insert menu
  • you right-click a database field and choose the Insert Grand Total command from the shortcut menu that appears.

Use the Insert Grand Total dialog box to select the kind of grand total you want to appear on your report.

Note:    You cannot sum or average a text, Boolean, or date field.

Summary Function

Select the summary function that you want to use for the grand total.
