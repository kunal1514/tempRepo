Enabling server-side processing

  1. On the File menu, click Report Options.
  2. Select Perform Grouping on Server on the Report Options dialog box.

    This check box is inactive if Use Indexes or Server for Speed is not selected.

  3. Click OK.

Note:    You can also enable or disable this option quickly by selecting or deselecting Perform Grouping on Server on the Database menu as needed. If Use Indexes or Server for Speed is not selected in the Report Options dialog box, this command is inactive.
