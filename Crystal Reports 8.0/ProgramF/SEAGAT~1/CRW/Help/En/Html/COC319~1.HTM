Using the Group command

  1. Choose the Group command from the Edit Selection Formula menu, on the Report menu. The Group Selection Formula Editor appears. This formula restricts the groups that are printed.
  2. You can use the full range of operator, functions, and data fields to edit your formula.
  3. Click Save and Close when finished. The program limits the report to the groups specified.

Note:    

  • No object frame appears when you create a selection formula; the formula is simply stored in the report.
  • Your group selection formula must be Boolean, that is, it must result in a Yes or No answer.
  • You cannot use commas in numbers you enter in a formula. Enter only the number itself.
