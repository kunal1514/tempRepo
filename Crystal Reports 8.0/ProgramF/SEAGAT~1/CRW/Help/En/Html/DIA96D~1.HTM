Format Box tab (Format Editor)

The Format Box tab appears in the Format Editor when you right-click a box that you have created and choose Format Box from the shortcut menu that appears. You can also select this option from the Format menu.

Use the Format Box tab to change the fill, border, width and style of your box.

Style (Border)

This list shows all the available line styles for your box; such as: single, double, dashed, and so on.

Width (Border) options

Click the button which represents your desired border line width.

Color (Border)

This list shows all the available colors for your box. Select the desired color from this list.

Drop Shadow

Select this check box if you want to have a drop shadow below and to the right of your box.

Color (Fill)

If you select this check box, the Color list appears showing all the available colors to fill your box.

Close Border on Page Breaks

Select this check box if you want the program to convert box segments into finished boxes when the program splits a box due to a line break. For example, if you have a one inch square box which is split in half, the segment on the first page is missing a bottom, the segment on the second page is missing a top. When this property is selected, the program adds the bottom to the first box and the top to the second, creating a finished half inch box on each page.

Extend to Bottom of Section when Printing

Select this check box if you want the program to allow the box to extend to the bottom of the section.

Suppress

Select this check box if you do not want the box to print.
