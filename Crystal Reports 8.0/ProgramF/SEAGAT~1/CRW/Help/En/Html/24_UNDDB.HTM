Understanding Databases

This section describes the essentials you'll need to know to understand database design and use. After describing relational databases (including those from ODBC data sources), indexing, and table linking, the section introduces the concepts of server-side processing (pushing report processing to the server level) and field mapping (re-establishing report and database field mappings after a database changes). Finally, the section focuses on common database-related tasks such as changing a database driver and creating an ODBC data source.
