Using the Auto-Arrange Report command

  1. Create your report as you normally would.
  • Select and position database fields.
  • Build and place the database formulas.
  1. Group and summarize the data using groups and summary fields.
  2. On the Format menu, click Auto-Arrange Report. The program adjusts field size and placement, and automatically prints an attractive, balanced report to the screen.
