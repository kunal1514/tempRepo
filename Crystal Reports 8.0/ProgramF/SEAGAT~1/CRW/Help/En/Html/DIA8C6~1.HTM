Set Location dialog box

This dialog box appears when you choose the Set Location command from the Database menu.

Use the Set Location dialog box to specify a new database or a new location for the database used in the report.

Databases

List the aliases used in the current report. The first alias on the list is selected.

Location

This information box at the bottom of the dialog box displays the referenced table, server type, server name, database and user ID of the selected alias.

Set Location

Click this button to specify a different location or file for the selected database. When you click this button, the Data Explorer appears.

Same As Report

Click this button to set the location of the selected database to the same path as the report.

Convert to UNC

Click this button to convert the database field path to a universal naming convention format.
