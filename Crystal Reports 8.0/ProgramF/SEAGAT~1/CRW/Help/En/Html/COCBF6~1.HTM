Standard toolbar

The standard toolbar contains several commonly used commands. You can switch the toolbars on and off using the Toolbars command, or from the shortcut menu that appears when you right-click the toolbar or menu area.

If you are not sure about the function of a button on the Standard toolbar, place the pointer over the button and hold it there until the tool tip appears.

The buttons on the Standard toolbar perform the following functions:


Creates a new report. Shortcut to the New command.


Opens an existing report. Shortcut to the Open command.


Saves the report. Shortcut to the Save command.


Sends the report to a printer. Shortcut to the Printer command.


Displays a preview of the report on the Preview tab. Shortcut to the Print Preview command.


Exports the report in one of several popular formats. You can export to a disk file, to an application, to a Lotus Notes database, to an Exchange Folder, or to e-mail. Shortcut to the Export command.


Refreshes report data. Shortcut to the Refresh Report Data command.


Cuts selected object/data to the clipboard. Shortcut to the Cut command.


Copies the selected object/data to the clipboard. Shortcut to the Copy command.


Pastes object/data from the clipboard to the report. Shortcut to the Paste command.



Undoes an action. Shortcut to the Undo command. Use the drop down list to undo more than one action.



Redoes the latest action that was undone. Shortcut to the Redo command. Use the drop down list to redo more than one action.



Inserts a hyperlink. Shortcut to the Hyperlink command on the Insert menu.



Opens the Field Explorer for you to insert database fields Shortcut to the Database Field command on the Insert menu.



Inserts a text object. Shortcut to the Text Object command on the Insert menu.


Inserts a summary. Shortcut to the Summary command on the Insert menu.



Activates the Report Expert. Shortcut to the Report Expert command.



Activates the Section Expert for you to format any section of the report. Shortcut to the Section command.



Activates the Select Expert for you to set record/group selection criteria. Shortcut to the Select Expert command.



Sets record sort order. Shortcut to the Record Sort Order dialog box.



Inserts a chart. Shortcut to the Chart command on the Insert menu.



Inserts a map. Shortcut to the Map command on the Insert menu.



Finds a specific record. Shortcut to the Find command.



Sets the magnification factor for viewing your report.
