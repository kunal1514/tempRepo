Highlighting Expert command

Use the Highlighting Expert command to display the Highlighting Expert to apply conditional formatting to currency and number fields.

Use the Highlighting Expert to highlight field values that are of particular interest. For example, to highlight customers whose the sales are greater than $25,000, you could use the Highlighting Expert to apply red background color to the {customer.LAST YEAR'S SALES} field, if the value is greater than $25,000.

You can also use the dialog box to set conditional font color and border style.
