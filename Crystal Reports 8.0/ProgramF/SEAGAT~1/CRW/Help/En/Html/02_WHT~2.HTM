Formatting objects, images, and symbols

Use accounting formats

Control how the negative symbol, currency symbol, and zero value are displayed in a report. You can also reverse the sign for debit and credit.

Format multiple objects

Select multiple objects, including text, field, chart, map, bitmap, OLAP grid, Cross-Tab, and OLE objects, to move, align, size, and copy and paste together.

Choose from more image formats

Seagate Crystal Reports now supports more image formats for blob fields including JPEG, DIB, PNG, and TIFF.

Insert pre-formatted HTML or RTF data into reports

Insert pre-formatted data in RTF or HTML into a report. The Seagate Crystal Report Designer will extract any information about the font, size, style, color and paragraph and display the data with the specified attributes.
