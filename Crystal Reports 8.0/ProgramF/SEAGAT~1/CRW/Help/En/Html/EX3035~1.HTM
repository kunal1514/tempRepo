Rows/Columns tab (OLAP)

The Rows/Columns tab allows you to choose dimensions and fields for display in the grid.

Dimensions

Use this list to select dimensions for the object.

Columns

This list displays the column dimensions.

Rows

This list displays the row dimensions.

Add Column

Use this button to add a selected dimension to a column.

Add Row

Use this button to add a selected dimension to a row.

Remove

Use this button to remove a selected dimension from either the row or column dimension.

Use Long Field Names

Use this check box to include long field names in your OLAP Grid object. If you clear this check box, the program will use short field names, such as PR01 and PR02 instead of Racing Bicycles and Mountain Bikes.
