Options command

Use the Options command to modify settings that apply to the reporting environment. Using these settings, you can customize the program to fit the way you work.

This command takes you to the Options dialog box. This dialog box serves as a control panel for configuring the program in a number of different areas.

Once there you can change default settings in the following tabs:

  • Layout tab
  • New Report tab
  • Database tab
  • Editors tab
  • Data Source Defaults tab
  • Reporting tab
  • Fields tab
  • Fonts tab

Your settings here affect new reports and fields added to existing reports, but they do not affect the work done before you changed the settings.
