Running Total Fields

Use the Running Total Fields branch of the tree as a gateway for editing, renaming, deleting, creating new, or inserting Running Total fields into your report.

The Running Total Fields list displays the names of all the Running Total fields currently in your report. You insert Running Total fields by dragging them into the report, or by selecting them and then clicking Insert to Report.

Each Running Total field added to the report has a check mark added in front of its name in the Field Explorer.

Insert to Report

Use this button to insert the Running Total field selected in the Running Total Fields list into your report.

New

This button opens the Create Running Total Field dialog box where you can specify the details for your Running Total. This option is available from the root of the Running Total Fields branch so you can create a new Running Total for your report even when there are none existing.

To add a new Running Total Field, add a name, field to summarize, and Evaluate/Reset criteria in the Create Running Total Field dialog box and click OK.

Edit

This button opens the Edit Running Total Field dialog box and displays the currently selected Running Total for you to edit.

Rename

This button lets you rename the Running Total you select in the Running Total Fields list. Running Total Fields can be renamed in the standard Windows fashion.

Delete

This button deletes the Running Total field selected in the Running Total Fields list. If the Running Total is already on the report, you are asked to confirm the deletion as it will also be deleted from the report.
