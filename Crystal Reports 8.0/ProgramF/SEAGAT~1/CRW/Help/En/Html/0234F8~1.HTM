Text formatting

Change fonts conditionally

In addition to conditionally formatting text by style and size, you can now conditionally format database fields by font.

Choose from more date, time, and date-time formats

Choose from a list of sample styles for date, time, and date-time formats or further customize these styles to suit your reporting needs.

Similar enhancements have also been made to numbers and currencies.

Rotate text objects

Rotate text objects, memo fields, and string fields 90 or 270 degrees.

Control indentation for lines

Specify the indentation for lines in string fields, memo fields, and text objects.

Specify a fractional point size

Use a fractional point size for a database field or a text object.

Control line and character spacing

Specify the spacing between lines and characters for text objects, string fields, and memo fields.

Use the search function to locate text quickly within reports

In the Preview tab, search for strings in text objects, field objects, subreports, Cross-Tabs, and OLAP grids.

Add new shapes

Insert new shapes such as circles, ovals, and boxes with rounded corners.
