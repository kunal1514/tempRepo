Using the Group Name Field command

  1. Choose the Group Name Field command from the Insert menu. The Field Explorer appears with the Others selected.
  2. Highlight the desired Group Name Field and click the Insert button. An object frame appears.
  3. Drag the Group Name field to the desired insertion point and click to place it.
  4. Repeat Steps 2 and 3 to add any additional Group Name fields.
