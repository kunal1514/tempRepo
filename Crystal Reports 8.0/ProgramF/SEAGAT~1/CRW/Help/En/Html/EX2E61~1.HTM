Report Expert options

Design Report

When you are finished designing the report, you can click the Design Report button to view the report template in the Design tab of the Seagate Crystal Reports window. By viewing the report in the Design tab, instead of the Preview tab, you avoid immediately requesting data from the database. This option is helpful for Power users that want to quickly customize the look of the report without waiting for the program to obtain all of the report data. This is also useful if the report is based on a very large database and takes a long time to collect and display all data in the Preview tab.

Preview Sample

Click the Preview Sample button to display a report using either all the data from the database or only a small selection of data. The report is generated and displayed in the Preview tab. This option is useful when you want to design and work with the report layout without having to wait for all data to be generated. Some reports can take long periods of time to be generated, forcing you to wait to preview and edit the layout. By previewing only a sample of the entire database, you save time and effort creating your report.
