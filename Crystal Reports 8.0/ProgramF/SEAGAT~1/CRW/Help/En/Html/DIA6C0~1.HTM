Save As dialog box

This dialog box appears when you choose the Save As command from the File menu, you are working on a new report and have not yet saved it and use this dialog box to specify the name (and path) under which you want to save your report file, or you choose the Save command from the File menu while working on a new report that you have not yet saved.

Use the Save As dialog box if you have made changes to an existing report file and you want to save the changed version of your file under a different name. All changes you have made while working on the report will be saved to the new file and your original report file will remain unchanged.

Specify the name, location and type of the file that you are saving.
