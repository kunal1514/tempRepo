Customize Style tab (OLAP and Cross-Tab)

Use the Customize Style tab to format a grid with background color, gridlines and cell margins.

Note:    The Customize Style tab only formats the primary grid dimensions you have selected in the Rows/Columns tab.

Columns

This list displays the dimension and levels used as the column aspect in the grid.

Rows

This list displays the dimension and levels used as the row aspect in the grid.

Background Color

Use this list to select a color for the selected row or column level.

Alias for formulas

Type the alias for the selected dimension. Shortened dimension names can be useful when building conditional formatting formulas.

Show Cell Margins

Select this check box to expand column widths to show all the grid. If you clear the checkbox, column widths will be uniform and some data will be hidden behind the grid.

Indent Row Labels

Select this check box then type the indentation measurement for row labels in the box. This can be as long as two pages.

When you specify an indentation, all row labels, except the label for the input member of the first level will be indented by the same amount.

Repeat Row Labels

Select this check box to repeat row labels on every page for grids that print across more than one page.

Keep Columns Together

Select this check box to keep columns together on every page for grids that print across more than one page.

Row totals on Top

Select this check box to display totals on the top (relative to the corresponding row input member).

Column totals on Left

Select this check box to display place column totals on the left (relative to the corresponding column input member).

Suppress Empty Rows

Select this check box to suppress empty rows for the OLAP Grid object. If you clear the checkbox, empty rows will be shown in the OLAP Grid object.

Suppress Empty Columns

Select this check box to suppress empty columns for the OLAP Grid object. If you clear the checkbox, empty columns will be shown in the OLAP Grid object.

Format grid lines

Click this button to specify settings for the grid lines in the Format grid lines dialog box.

Type the amount of space between the left edge of the report page and the label in the Horizontal gap box.
