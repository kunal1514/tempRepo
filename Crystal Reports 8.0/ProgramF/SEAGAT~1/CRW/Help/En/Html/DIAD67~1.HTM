Format Other Dimensions dialog box

This dialog box appears when you click the Other Dimensions button on the Data tab of the Map Expert or OLAP Report Expert. Note that this button is available when you click the OLAP layout button, which appears if you are connected to an OLAP data source.

Use the Format Other Dimensions dialog box to set field values for your other dimensions.

Other Dimensions

This list displays the remaining dimensions in the OLAP Grid. Use the list to select the dimensions for which you want to set the field values.

Select a field value

Use this button to select the fields you want displayed in an OLAP Grid object, for the dimension you have selected in the Other Dimensions list. When you click this button, the Select a Field for Dimension dialog box appears.

The field value you select in that dialog box will appear in the list to the right of the other dimension name.
