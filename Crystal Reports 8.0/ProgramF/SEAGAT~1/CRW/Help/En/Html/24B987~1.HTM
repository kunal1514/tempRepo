Server Type

Mailbox Admin

To log on to the Mailbox Admin server type
  1. Open the Data Explorer dialog box by choosing:
      • Log On/Off Server from the File menu (when there is no report opened),
      • Log On/Off Server from the Database menu (when there is a report opened).
  2. Select Mailbox Admin from the More Data Sources folder.
  3. Double-click the folder or double-click Make a New Connection. The Logon to Exchange Servers dialog box appears.
  4. Click Select Profiles and choose a profile from the list on the Select Profiles dialog box.
  5. Click OK to return to the Logon to Exchange Servers dialog box.
  6. Click OK once more to return to the Data Explorer dialog box.
To set the location of a table accessing a Mailbox Admin server type
  1. On the Database menu, click Set Location.
  2. When the Set Location dialog box appears, click Set Location.
  3. Find and highlight the new Mailbox Admin table you want to use in the Data Explorer dialog box and click Set when done.

    Verify that the new table has been selected by reviewing the Table, Server Type, Server Name, Database (alias name displayed), and User ID in the Location section of the Set Location dialog box.

    Note:    The value MAPI PROFILES is always displayed as the Database name for this server type.

  4. Click Done after reviewing the table information.
