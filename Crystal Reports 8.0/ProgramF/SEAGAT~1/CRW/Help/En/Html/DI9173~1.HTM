Cross-Tab Group Options dialog box

This dialog box appears when you click Group Options in the Format Cross-Tab dialog box.

Use the Cross-Tab Group Options dialog box to create or modify group options for any row or column field used in the cross-tab report. The Group Options button is not available for summarized fields.

Things to consider when grouping on Date fields

When your groups are based on a Date field, the program activates two additional list boxes. The conditions in these boxes enable you to further define sorting and grouping specifications and display specifications.

Related topics

Format Cross-Tab dialog box

Common tab

Use the Common Tab to select the field you want to group on and specify the sort order for.

Sort Order

This list shows the sort direction options. The default option is in ascending order. If you want to change the sort direction, click the box to reveal other options and make your selection from the list.

Note:    The additional lists for a Date field include choices for grouping by time periods (each week, each month, and so on) and selecting whether to display the first or last date in a period.

Customize Group Name Field

Use this check box if you want to display a name different from the one chosen by default. By default, the group name on the cross-tab is the same as that of the of the column or row you are grouping on. Selecting this check box makes all options below it active.

Choose From Existing Field

Select this option to create a customized group name from an exiting database field. For example, your cross-tab may be grouped on Customer ID, but you would rather display the Customer Name.

Use a Formula as Group Name

Select this option to create a customized group name from a formula. Clicking the formula button opens the Group Name Formula Editor.

Format Cross-Tab dialog box

Specified Order tab

The Specified Order tab appears whenever you select in specified order as a sort order from the Insert Group dialog box or the Change Group Options dialog box.

Use the Specified Order tab when you want to set up custom groups. Custom groups are freeform groups that are not necessarily triggered by any one field value. You define the groups and you specify the individual values that belong in each.

Named Group

Use this box to enter the name you want for your specified order group.You can also browse and select values in the database field to use as group names. For example, if you are creating specified groups to represent regions in the USA, you might look at the list and choose Alaska and Hawaii to be regions of their own.

The box below displays the fields you have selected and the groups you have created as specified order groups.

Up arrow

Click this button to move the highlighted group higher in the list.

Down arrow

Click this button to move the highlighted group lower in the list.

New

Click this button to open the Define Named Group dialog box where you can create a new named group. If you entered a new name in the Named Group list on the Specified Order tab, it appears as the Group Name on the Define Named Group dialog box.

Edit

Click this button open the Define Named Group dialog box where you can edit a group you've highlighted in the Named Group list.

Delete

Click this button to delete the highlighted group.

Format Cross-Tab dialog box

Others tab

The Others tab appears whenever you create a new Named Group on the Specified Order tab of the Cross-Tab Group Options dialog box.

Use the Others tab to specify what should be done with database fields that are not chosen as part of your specified order groups.

Discard all others

Select this option if you do not want to print the unselected fields in the groups for your Cross-Tab. If you summarize groups, values in the discarded groups will not be summarized nor will they be included in any grand total if you have one.

Put them all together, with the name

Select this option and enter a name if you want to put all unselected fields in one group. The default name for this group is Others. If you summarize groups, the values in the group will be summarized and will be included in any grand total.

Leave in their own groups

Select this option if you want the unselected fields to form their own groups. If you summarized groups, the values in the group will be summarized and will be included in any grand total.
