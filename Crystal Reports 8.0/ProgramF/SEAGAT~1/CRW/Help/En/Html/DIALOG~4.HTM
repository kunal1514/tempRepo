Layer Control dialog box

The Layer Control dialog box appears when you:

  • choose the Change Map Layers command from the Analyzer menu
  • select Change Map Layers from the shortcut menu on the Analyzer tab.

Use this dialog box to add, remove or change layers on your map. For example, you could rearrange the layers so that the Major Cities layer appears above the Highway layer, or so that the Capitals layer appears above the Top 20 Cities layer.

Note:    It may be necessary to zoom in on the map to view layer changes.

Layers

The Layers list contains the layers options for your map.

Up

Use the Up button to move a layer up in the Layers list.

Down

Use the Down button to move a layer down in the Layers list.

Add

Use the Add button to add a layer to the Layers list.

Remove

Use the Remove button to remove a layer from the Layers list.

Visible

Use the Visible check box to make a layer visible on the map. If this check box is not selected, the selected layer will be hidden.

Automatic Labels

Use the Automatic Labels check box to specify whether the layer is labeled automatically on the map.

Display

Click the Display button to bring up the Display Properties dialog box. The Display Properties dialog box allows you to further manipulate the mode and zoom layering of a selected layer.
