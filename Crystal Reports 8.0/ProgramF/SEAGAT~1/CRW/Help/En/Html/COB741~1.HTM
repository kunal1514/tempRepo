Format Object/Expert commands

Note:    The formatting command that you see on the Format menu and the Analyzer menu changes to match the object you have selected (for example, Format Field, Format Box, Format Graphic, Map Expert, Chart Expert and so on).

Use the displayed command to format the selected object.

The Format commands open the Format Editor, which presents you with the various formatting properties that you can apply to the object. The properties vary for each kind of object, but they include:

  • applying borders and colors
  • changing the font for text or numbers
  • suppressing the object so that it does not print
  • cropping the size of the object
  • maintaining the object size or letting it grow.

If you selected a map, this command changes to the Map Expert command. Use the Map Expert to edit or format a map.

If you selected a chart, this command changes to the Chart Expert command. Use the Chart Expert to edit or format a chart.
