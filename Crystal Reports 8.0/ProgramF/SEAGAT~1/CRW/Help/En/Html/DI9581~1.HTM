Find dialog box

The Find dialog box allows you to locate text in a report. This dialog box appears when you choose Find from the Edit menu.

Find what

Type the text you want to search for, paste it from the clipboard or click a recent entry from the list.

Find Next

Click this button to find and select the next occurrence of the text specified in the Find what box.

Match case

Select or clear this check box to control whether the exact sequence of upper and lower case letters in the search text should be considered.

Match whole word only

Select or clear this check box to control whether only whole words are to be found instead of searching for text within words.

Direction

Specifies which direction to search starting from the current cursor position. Click Up to search toward to start of the report or Down to search toward the end of the report.

Advanced Find

Click this button to call the Search Expert.

This button is only available in the Preview tab.
