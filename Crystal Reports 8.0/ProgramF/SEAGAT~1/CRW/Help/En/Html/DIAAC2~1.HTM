Welcome to Seagate Reports dialog box

This dialog box appears whenever you run Seagate Crystal Reports, unless you clear the Show welcome dialog at startup check box. You can also open this dialog box by choosing the Welcome Dialog command from the Help menu.

Use the Welcome to Seagate Reports dialog box to specify whether you would like to create a new report from scratch, open an existing report, or cancel the action.

Using the Report Expert

Click this button to create a new report using one of the report experts. The Seagate Crystal Report Gallery appears where you can select an expert to help you create your report.

Asa blank report

Click this button to create a new report from scratch. Choose the data on which to base your report from the Data Explorer.

Open an existing report

Click this button to open an existing report. An Open dialog box will appear where you can select the desired report and click OK to open it.

Show welcome dialog at startup

Select this option to have Seagate Crystal Reports show the Welcome dialog on startup.

If this option is not selected, the Welcome dialog will not appear on startup. Clear this option to have Seagate Crystal Reports show the Welcome dialog each time on startup.
