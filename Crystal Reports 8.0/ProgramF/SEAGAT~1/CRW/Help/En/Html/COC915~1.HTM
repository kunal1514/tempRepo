Sort Records command

Use the Sort Records command to change the order in which records appear in your report. You can add and remove sort fields and you can define the sort order (ascending or descending) for the data in your report. You can not, however, remove group sorts with this command.

When you choose this command, the Record Sort Order dialog box appears.

Note:    This command is also available as a button on the Standard toolbar.
