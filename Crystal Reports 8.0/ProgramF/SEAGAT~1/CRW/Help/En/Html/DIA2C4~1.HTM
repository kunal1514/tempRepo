Options dialog box

This dialog box appears when you choose the Options command from the File menu.

Use the Options dialog box to configure the program—that is, to set up the program so that it works like you do. This dialog box offers a wide range of options for helping you work faster and easier.

Note:    Many of these options also appear in the Report Options dialog box. Use the Options dialog box to change the default values for the program, use the Report Options dialog box to change the options for the current report.
