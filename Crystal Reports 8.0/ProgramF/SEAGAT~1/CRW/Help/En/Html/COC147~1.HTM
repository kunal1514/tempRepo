Insert menu

The Insert menu includes commands used to insert database fields, text objects, formula fields, parameter fields, cross-tab objects, subtotals, grand totals, summaries (counts, averages, and so on), groups, and sections. This menu also includes commands that enable you to insert group name fields, subreports, graphics, lines, boxes, charts, and OLE objects into a report.
