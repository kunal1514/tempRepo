Specifying records to be included in a query

  1. From the Database Fields area, click the field that contains the data you want to filter or specify.
  2. Click Add.

    The field is added to the Select Fields area.

  3. Use the record selection drop-down lists to specify the filter/selection criteria to be applied to the field.
  4. Repeat Steps 2 and 3 for every field in the Select Fields area. This will increase your report performance as only the records matching the criteria you have specified are included.

  5. Click Next.

    The SQL tab appears.
