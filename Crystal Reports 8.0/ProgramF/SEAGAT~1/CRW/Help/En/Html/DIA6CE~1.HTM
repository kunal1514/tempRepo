Seagate Crystal Report Gallery

This dialog box appears when you choose the New command from the File menu.

Use the Seagate Crystal Report Gallery to select a Report Expert for step-by-step help in creating a report, or create a report from scratch.

If you select the As a blank report option the Data Explorer appears.

You can select one of the following experts:

  • Standard

    The Standard Report Expert guides the creation of a typical report.

  • Form Letter

    The Form Letter Report Expert guides the creation of a letter that combines standard text with customer data.

  • Form

    The Form Report Expert guides the a report with letterhead or log graphics. This is commonly used to create invoices.

  • Cross-Tab

    The Cross-Tab Report Expert guides the creation of a report that contains a summarized grid.

  • Subreport

    The Subreport Report Expert guides the creation of a report that contains a second report. You can either include an existing report or create one.

  • Mail Label

    The Mail Label Report Expert guides the creation of a report with multiple columns. This is commonly used to create a sheet of address labels.

  • Drill Down

    The Drill Down Report Expert guides the creation of a report that displays summarized information and reveals the details when you drill down.

  • OLAP

    The OLAP Report Expert guides the creation of a report that contains a summarized grid based on an OLAP data source.

Note:    The Add Index button on each Report Expert opens a standard Windows search dialog box. Use the Attach New Index dialog box to search for the index you'd like to add.
