Copy command

Use the Copy command any time you wish to hold a copy of a selected object on the Clipboard for later use without disturbing the object as it currently appears in the report.

Note:    The Copy command can be used in the Formula Editor (for any formula element) via the Windows keyboard command Ctrl-C.
