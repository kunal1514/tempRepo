Hyperlink command

Use the Hyperlink command to create a hyperlink from the selected object to to a web site, a local HTML file, a field value, an e-mail address, or another Seagate Crystal Report file.

The hyperlink is saved with your report and is available to other users as a way of viewing additional information.

When you choose the Hyperlink command, the Hyperlink tab (Format Editor) of the Format Editor appears.

Note:    

  • If you create a hyperlink to another Seagate Crystal Report file, you are creating an on-demand subreport. Just as one subreport cannot contain another subreport, you cannot create a hyperlink between two subreports. See How subreport linking works.
  • To select a hyperlinked object, use Shift-click or Ctrl-click. Clicking on a hyperlink, without using the Shift or Ctrl key, activates the link.
  • This command is also available as a button on the Standard toolbar.
