Supplementary Tools toolbar

The Supplementary toolbar provides quick access to many advanced, report-enhancing experts and features. To display this toolbar click the Toggle Supplementary Toolbar button, found on the far-right end of the Formatting toolbar.

The buttons on the Supplementary toolbar perform the following functions:

Inserts a line. Shortcut to the Line command on the Insert menu.

Inserts a box. Shortcut to the Box command on the Insert menu.

Inserts a picture. Shortcut to the Picture command on the Insert menu.

Arranges report elements automatically to best fit the page. Shortcut to the Auto-Arrange Report command.

Activates the Report Style Expert to apply a professionally designed style to a report. Shortcut to the Report Style Expert command.

Inserts a group. Shortcut to the Group command on the Insert menu.

Inserts a subreport. Shortcut to the Subreport command on the Insert menu.

Inserts a cross-tab object. Shortcut to the Cross-Tab command on the Insert menu.

Finds Top/Bottom N records or sorts on summary information. Shortcut to the Top N/Sort Group Expert command.


Activates the appropriate editor to modify object properties.


Opens the Visual Linking Expert to define links between tables. Shortcut to the Visual Linking Expert command.
