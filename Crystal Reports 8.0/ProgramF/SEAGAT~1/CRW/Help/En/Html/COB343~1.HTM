Parameter Field command

Use the Parameter Field command to create or insert a parameter (prompting) field into the report itself, into a formula, into a record selection formula or into the body of the report. When using a parameter field, the program prompts you for a value when you first run the report and whenever you refresh the data.

The program also allows you to edit, delete, or rename existing parameter fields with this command.

When you choose the Parameter Field command, the Field Explorer appears with Parameter Fields selected.

Note:    You can also open the Field Explorer by clicking the Insert Fields button on the Standard toolbar.
