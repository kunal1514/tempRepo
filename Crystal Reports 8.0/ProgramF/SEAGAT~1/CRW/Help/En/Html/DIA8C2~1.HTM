Set Print Date/Time dialog box

This dialog box appears when you choose the Set Print Date/Time command from the Report menu.

Use the Set Print Date/Time dialog box to set the date for your report. You can choose either today's date or specify another date.

Today's Date/Time

Select this check box to use today's date and time as your report date.

Other

Select this check box if you want to use a date other than today's as the report date. When you select this check box, the Year, Month, and Day edit boxes are activated where you can specify the desired print date.

Date

Use the Date list to select a future print date.

Time

Use the Time box to set the print time.
