Show/Hide Sections command

Use the Show/Hide Sections command display the Section Expert, in which you can select report sections to show or hide using the Suppress or Hide options.

  • If you select Hide (Drill-Down OK) for a section, you will only see the information in that section if you drill down. You can print the information in the section once you have drilled down to it.
  • If you select Suppress (No Drill Down) for a section, you will not be able to see the information in that section as long as it is suppressed. You cannot drill down on a suppressed section.
