Report Expert

The Report Expert appears when you click the New button from the Standard toolbar and then choose one of the built-in Report Experts from the Seagate Crystal Report Gallery or when you choose the Report Expert command from the Analyzer menu.

Each Report Expert consists of several tabs that lead you through step-by-step instructions for creating the specified report.

There are eight Report Experts:

  • Standard Report Expert
  • Form Letter Report Expert
  • Form Report Expert
  • Cross-Tab Report Expert
  • Subreport Expert
  • Mail Label Report Expert
  • Drill Down Report Expert
  • OLAP Report Expert.

Each Expert guides you through the creation of a report by providing a series of tabs. Many of the experts have tabs unique to a specific type of report. For example, the Mail Label Expert has a tab that allows you to specify the type of mailing label you want to use.
