What's new in Crystal syntax

Many new features have been added to Crystal syntax for Seagate Crystal Reports Version 8.

Note:    Equivalent versions of these features are also supported by Basic syntax.

New control structures

Crystal syntax supports three new controls structures: the Select expression, the While loop, and the For loop. Select is a good alternative to the If expression when you want to use the result of a single expression to select from a number of alternatives. The While and For loops introduce looping into Seagate Crystal Reports. Looping is useful when you want to evaluate some expressions in your formula multiple times. A typical application is to extract information from string database fields for custom formatting in a report. In this case, you loop through the characters of the string field.
