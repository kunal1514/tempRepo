Layout tab

Use the Layout tab to set the default grid, ruler, preview page, and object format options.

View options

Show Rulers in Design

When you select this check box, the Ruler is displayed on the Design tab. This option is selected by default.

Show Rulers in Preview

When you select this check box, the Ruler is displayed on the Preview tab. This option is selected by default.

Show Guidelines in Design

When you select this check box, the program displays guidelines on the Design tab. This option is not selected by default.

Show Guidelines in Preview

When you select this check box, the program displays guidelines on the Preview tab. This option is not selected by default.

Show Hidden Sections in Design

When you select this check box, hidden and suppressed sections in your report are displayed on the Design tab. Report objects in these sections are shown on a gray background. This option is selected by default.

Show Short Section Names in Design

When you select this check box, the program abbreviates the section names that appear in the gray area on the left side of the Design tab (the gray area is reduced in size). This adds extra width to the report page and allows you to work with more of your report on the screen.

When you clear this check box, the program displays full section names and maintains the gray area at full width.

This option is selected by default.

Show Section Names in Preview

When you select this check box, the program displays abbreviations of the section names in a gray area on the left side of the Preview tab.

When you clear this check box, no section names are displayed and the entire width of the Preview tab is used to display the report page.

This option is selected by default.

Show Page Breaks in Wide Pages

When you select this check box, indicators are displayed on the Preview tab for page breaks in wide pages. This option is selected by default.

Show Tool Tips in Design

When you select this check box, the program displays tool tips when you are working on the Design tab. Tool tips show the name of a button when you leave your mouse pointer over it for a few seconds. This option is selected by default.

Show Tool Tips in Preview

When you select this check box, the program displays tool tips when you are working on the Preview tab. Tool tips show the name of a button when you leave your mouse pointer over it for a few seconds. This option is selected by default.
