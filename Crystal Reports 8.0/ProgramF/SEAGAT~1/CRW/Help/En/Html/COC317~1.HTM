Using the Border and Colors command

  1. Select the field you want to format.
  2. Choose the Border and Colors command from the Format menu (or right-click and choose the Change Border command from the shortcut menu that appears). The Format Editor appears with the Border tab (Format Editor) selected.
  3. Select the text color, fill, border, and box sizes you want.

Note:    All colors are fixed; they cannot be edited.
