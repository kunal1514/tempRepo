Viewing guidelines

You can view guidelines on the Design and Preview tabs by selecting the view options in the Options dialog box.

To view guidelines
  1. On the File menu, click Options.

    The Options dialog box appears.

  2. On the Layout tab, in the View options area, select the Show Guidelines in Design check box and/or the Show Guidelines in Preview check box.
  3. Click OK to save your changes.
