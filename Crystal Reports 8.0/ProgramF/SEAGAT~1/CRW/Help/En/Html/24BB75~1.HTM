The Visual Linking Expert

The Visual Linking Expert lets you easily link two or more tables. When you choose Add Database To Report from the Database menu and select an additional database table, the Visual Linking Expert appears and displays that database table.

The easiest way to link database tables is to select Smart Linking in the Visual Linking Expert. Smart Linking automatically chooses links for your tables based on common fields in tables or indexed fields (if your database supports indexed fields).
