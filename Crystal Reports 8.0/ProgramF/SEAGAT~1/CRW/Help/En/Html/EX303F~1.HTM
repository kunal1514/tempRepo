Subreport Expert

The Subreport Expert lets you create a main report and a subreport at the same time. The expert offers all the Standard Report Expert's flexibility for creating the main report. On the additional Subreport tab, you can choose an existing report to use as the subreport, or you can create a new report. New subreports are created by repeating the steps used in the Standard Report Expert.

The Subreport Expert has the following tabs:

  • Data tab
  • Fields tab
  • Group tab
  • Total tab
  • Top N tab
  • Chart tab
  • Select tab
  • Subreport tab
  • Style tab
