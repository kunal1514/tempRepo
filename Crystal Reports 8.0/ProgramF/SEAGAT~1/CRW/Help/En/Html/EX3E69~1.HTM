Group tab

Use the Group tab to specify how fields will be grouped on the report. Records will be sorted by their values on the group by fields.

Available Fields

This list displays all of the fields available for use in your report.

Browse Data

Click this button to browse the data of the field that is selected in the Available Fields list.

Group By

This list displays all of the fields that will be used for grouping data.

Add ->

Click this button to add a selected field from the Available Fields list to the Group By list.

<- Remove

Click this button to remove a selected field from the Group By list.

Order

Use the Order buttons to move a selected field up or down in the Group By list. Fields that appear higher in the list are given a higher priority.

Sort Order

Click in ascending order to sort the values for the field selected in the Group By box in ascending (A to Z, 1 to 9) order.

Click in ascending order to sort the values for the field selected in the Group By box in descending (Z to A, 9 to 1) order.

Click in original order to leave the values for the field selected in the Group By box in their original order.

Click in specified order to specify exactly how the fields will be ordered.

These options are only available when fields are listed in the Group By list.

New Named Group

Click this button to create a new group in the Define Named Group dialog box.

This button is only available when in specified order is selected in the Sort Order list.

Field values

Lists the field values for the field selected in the Group By list. Click a value to create a new group based on the value.

This list is only available when in specified order is selected in the Sort Order list.

Edit Named Group

Click this button to change the settings for a group in the Define Named Group dialog box.

This button is only available when in specified order is selected in the Sort Order list.

Add

Type a group name in the list, then click this button to create it.

This button is only available when in specified order is selected in the Sort Order list.

Del

Click this button to delete the selected group.

This button is only available when in specified order is selected in the Sort Order list.

Others

Click this button to organize the values that do not belong to any group in the Others dialog box.

This button is only available when in specified order is selected in the Sort Order list and at least one group exists.
