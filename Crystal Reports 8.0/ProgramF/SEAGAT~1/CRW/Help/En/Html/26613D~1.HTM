Creating a new query using the Crystal SQL Designer

Accessing the Crystal SQL Designer

  1. On the Start menu, select Programs, select Seagate Crystal Reports, then click Crystal SQL Designer.

    Note:    The location of the Crystal SQL Designer may vary depending on location you specified during installation.

  2. On the File menu, click New.

    {bmc bt_new.gif}

    The New Query dialog box appears.

  3. Click Use SQL Expert.

The Create SQL Expert appears. This expert contains six tabs that will lead you step-by-step through the query creation process.

Note:    The Links tab only appears when two or more database tables are selected on the Tables tab.

Previewing data

Once you select at least one field for the query on the Fields tab, you can view the data retrieved by your query at any time, by clicking Preview Query at the bottom of the dialog box. After you are done viewing the query data, click Edit to return to the SQL Expert.
