Axes tab

The Axes tab lets you set the properties for each of the chart axes. The Axes tab changes depending upon the type of chart you are creating.

Major

Use the Major check boxes to determine whether major guidelines will be shown for the chart axes. There is a Major check box for Group axis, a Major check box for Series axis, a Major check box for Data axis, and a Major check box for the Data2 axis. Major check boxes are only active if the chart is a cross-tab or OLAP chart.

  • Select a Major check box to show the guideline for the corresponding axis.
  • Clear a Major check box to hide the guideline for the corresponding axis.
Minor

Use the Minor check boxes to determine whether major guidelines will be shown for the chart axes. There is a Minor check box for Group axis, a Minor check box for Series axis, a Minor check box for Data axis, and a Minor check box for the Data2 axis. Minor check boxes are only active if the chart is a cross-tab or OLAP chart.

  • Select a Minor check box to show the guideline for the corresponding axis.
  • Clear a Minor check box to hide the guideline for the corresponding axis.
Auto Scale

When you select one of the Auto Scale check boxes, the number of data axis labels is adjusted to ensure a clear display.

Auto Range

When you select one of the Auto Range check boxes, the data values for the corresponding axis are assigned a default range.

Min

Use the Min text boxes to enter the range minimum for the data values on the corresponding axes. There is a Min text box for the Data axis, and there is a Min text box for the Data2 axis. The second Min text box is only active if the chart is a cross-tab or OLAP chart.

If the corresponding Auto Range check box is selected, the Min text box is inactive.

Max

Use the Max text boxes to enter the range maximum for the data values on the corresponding axes. There is a Max text box for the Data axis, and there is a Max text box for the Data2 axis. The second Max text box is only active if the chart is a cross-tab or OLAP chart.

If the corresponding Auto Range check box is selected, the Max text box is inactive.

Number Format

Use the Number Format lists to select a number format for the data values on the corresponding axis. There is a Number Format list for the Data axis (y), and there is another for the Data2 axis. The second Number Format list is only active if the chart is a cross-tab or OLAP chart.

If the corresponding Auto Range check box is selected, the Number Format list is inactive.

Automatic

If you click one of the Automatic option buttons, the program automatically sets the number of divisions for the corresponding chart axis. There is an Automatic option for the Data axis, and there is an Automatic option for the Data2 axis. The second Automatic option button is only active if the chart is a cross-tab or OLAP chart.

Manual

Use the Manual option buttons to manually set the number of divisions for the corresponding axes. There is a Manual option for the Data axis, and there is a Manual option for the Data2 axis. The second Manual option button is only active if the chart is a cross-tab or OLAP chart.

When you click one of the Manual option buttons, the adjacent text box becomes active. Use the text box to enter the number of divisions.
