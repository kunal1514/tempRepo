Parameter Fields

Use the Parameter Fields branch of the tree as a gateway for editing, renaming, deleting, creating new, or inserting parameter fields into your report.

The Parameter Fields list displays the names of all the parameter fields currently in your report. You insert Parameter fields by dragging them into the report, or by selecting them and then clicking Insert to Report.

Each Parameter field added to the report has a check mark added in front of its name in the Field Explorer.

Insert to Report

Use this button to insert the Parameter field selected in the Parameter Fields list into your report.

New

This button opens the Create Parameter Field dialog box where you can specify the details for your parameter. This option is available from the root of the Parameter Fields branch so you can create a new parameter for your report even when there are no existing parameters.

To add a new Parameter Field, add a name, prompting text, and value type in the Create Parameter Field dialog box. After choosing options and setting default values, click OK.

Edit

This button opens the Edit Parameter Field dialog box and displays the currently selected parameter for you to edit.

Rename

This button lets you rename the parameter you select in the Parameter Fields list. Parameter Fields can be renamed in the standard Windows fashion.

Delete

This button deletes the parameter field selected in the Parameter Fields list. If the parameter is already on the report, you are asked to confirm the deletion as it will also be deleted from the report.

Move Up and Move Down buttons

Use the Move Up and Move Down buttons to change the order of parameter fields in the Parameter Fields list. These buttons become available as needed.
