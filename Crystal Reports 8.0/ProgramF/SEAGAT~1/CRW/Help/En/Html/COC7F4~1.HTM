Using the Section command

  1. Choose the Section command from the Format menu. The Section Expert appears. The Sections list in the dialog box lists all of the sections in the current report.
  • By default the Sections list contains five listings: the Report Header section, the Page Header section, the Details section, the Page Footer section, and the Report Footer section.
  • If you have added groups or subtotals, the program creates a group header section (Group header #n) and a group footer section (Group footer #n) for each unique group.
  • If you have added summaries, the program creates a group header section (Group header #n) and a group footer section (Group footer #n) for each summary.

    Note:    Group sections are numbered consecutively in the order in which they were created, and they appear on the list in the format: Group header #n:file.triggerfieldname or Group footer #n:file.triggerfieldname.

  • If you have added a grand total (sum, average, count, and so on), the program creates a Grand Total section to hold the value. Only one Grand Total section will appear on the list.
  1. Select the option(s) of interest, and click OK to finish formatting the section.

    Not all options are available for formatting all sections. For example, the New Page Before and New Page After options are not available when you format the Page Header section. The program already generates a page break before each Page Header section, and since a header can not stand alone on a page, there is no need for the Page Break After option.

For maximum efficiency in formatting sections, do the following
  1. Right-click the section of interest in the gray area at the left of the Design tab.
  2. Choose the Format Section command from the shortcut menu that appears. The program takes you directly to the Section Expert.
  3. Make your formatting selections as described above.
