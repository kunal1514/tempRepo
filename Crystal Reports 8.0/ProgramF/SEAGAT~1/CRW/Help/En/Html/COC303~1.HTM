Using the Page Number Field command

  1. Choose the Special Field command from the Insert menu. The Insert Fields dialog box appears with the Others tab selected. Select the Page Number command from the list that appears in the scroll box.
  2. Drag the frame to the desired insertion point and click to place it.

Note:    The Section command on the Format menu allows you to reset the page number for invoices, statements, and so on
