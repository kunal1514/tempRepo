Message Tracking Logs

Description

Access to all fields from the Exchange Server's Message Tracking Log.

When message tracking is enabled, each component handling mail records its activities in a log maintained by the system attendant on each server. The log becomes a trace of the processing of each message as each component receives, processes, and delivers it to the next component. By default, the Message Tracking Log are stored in the \\<machine name>\<share name>\exchsvr\tracking.log directory. A file is created for each day and its name has the format YYYYMMDD.log.

The report designer will interpret each entry in the log(s) as a record, and it provides accesses to all fields of the message tracking log. The definition of the data stored in these files can be found in Exchange Server's on-line Help.
