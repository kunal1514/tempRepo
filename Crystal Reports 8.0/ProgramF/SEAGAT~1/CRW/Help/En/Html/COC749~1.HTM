Change Line Height command

Use the Change Line Height command to modify the measurements that determine how text is placed between two lines.

Note:    This command is only available in a report section which is not using Free-form Placement. Cancel free-form placement using the Free-Form Placement check box on the Common tab in the Section Expert.

By adjusting the line height and descent, you will be able to position text on a pre-printed form in a manner that prevents the top of the text from touching a line, and that prevents the bottom of the text from touching a line.
