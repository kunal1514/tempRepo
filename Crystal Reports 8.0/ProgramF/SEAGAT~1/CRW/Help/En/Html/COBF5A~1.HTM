Close command

Use the Close command to close the current report while keeping the application active.

When you choose the Close command:

  • If you have already saved your work, the program closes the report.
  • If you have not yet saved your work, the program asks you if you want to save your changes before closing.
      • If you click Yes and you have already given your report a name, the program saves your changes under that name and closes the report.
      • If you click Yes and you have not already given your report a name, the Save As dialog box appears. Assign a name to your report. When you finish, the program saves your changes under the new name and closes the report.
      • If you click No, the program closes the report. All of your changes since you last saved are lost.
      • If you click Cancel, the program cancels the close request and returns you to the report.
