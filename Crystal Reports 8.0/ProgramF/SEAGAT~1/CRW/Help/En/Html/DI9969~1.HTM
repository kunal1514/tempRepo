Character Separated Values dialog box

This dialog box appears when you use the Export command to export a file in Character Separated Values format. From the Export dialog box, select the Character Separated Values from the Format list; the Character Separated Values dialog box appears.

Use this dialog box to specify the character(s) you want used to separate fields and to specify single or double quotes as a delimiter to surround non-numeric fields.

Separator

When you export a report to a character separated file, you need to specify the character that will be used to separate one field from the next (the field delimiter) in that file. Enter any keyboard character in this box and the program will use that character as a separator.

When you import the file into another application, that application will be able to separate the data into fields whenever it sees the character you specify.

Note:    It is best not to specify quotation marks as your separator, since they are already used to surround non-numeric fields.

Delimiter

When you export a report to a character separated file, the program encloses all non-numeric fields in quotation marks. You need to specify whether the program should use single (') or double (") quotes. Using the keyboard, type the kind of quotation marks you want in this box.
