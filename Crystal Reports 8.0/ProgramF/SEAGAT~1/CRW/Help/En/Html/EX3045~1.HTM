Labels tab (OLAP)

Use the Labels tab to define which dimensions are labeled and how they are displayed in the grid.

Unlabeled Dimensions

Displays the dimension that will not be labeled on the grid.

Labeled Dimensions

Displays the dimensions that will be labeled on the grid.

Move (< >)

Click these buttons to move selected dimensions between the Unlabeled and Labeled lists.

Move all (<< >>)

Click these buttons to move all the dimensions between the Unlabeled and Labeled lists.

Order

Click the up and down arrow buttons to change the order of the Labeled dimensions.

Label Position

Click Top to display labels at the top of the grid object.

Click Left to display labels to the left of the grid object.

Click Don't display labels to hide labels.

Label Spacing

Type the amount of space between the top edge of the grid object and the label in the Vertical gap box.
