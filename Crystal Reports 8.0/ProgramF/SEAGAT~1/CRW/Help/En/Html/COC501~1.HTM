Apply Changes to All Charts command

On the Analyzer tab, each change made to your chart is instance-specific. For example, if you rename the title of your chart from the Analyzer tab, the title change applies to only that chart instance. However, if you want to apply the title change to all chart instances, select the Apply Changes to All Charts command from the Analyzer menu.
