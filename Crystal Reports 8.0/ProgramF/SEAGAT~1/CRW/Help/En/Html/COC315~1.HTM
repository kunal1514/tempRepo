Using the Summary Info command

Use the Summary Info command to read or include the following information with your report: comments, author, keywords, title, subject and template information. Summary information accompanies the report but does not appear in the report itself.

  1. Choose the Summary Info command from the File menu. The Document Properties dialog box appears with the Summary tab active.
  2. Enter/edit the desired information.
  3. Click the Statistics tab to read document statistics
  4. Click OK to return to your report.

Note:    You can enter a Comment of any length up to one hundred lines. The text wraps automatically to fit the lines within the margins of the edit box.
