Graduated Option view

A Graduated map is much like a Ranged map in that it shows one symbol per instance of a specified item. Use a Graduated map for displaying data that is linked to points rather than precise geographical areas (as is the case with Ranged map types). For example, a map of an individual region could use graduated circles to represent the sales for each office. The size of each circle would be proportional to the sales (or to the sales range) of the office it represents. A Graduated map provides a more efficient representation of point data than a Ranged map does, and it uses sized symbols rather than colors to distinguish variations in the values of the items it represents.

You can choose the symbol you want to use on your Graduated map. Each symbol is proportional in size to the value of the item it represents (within a range of three sizes).

Symbol style

Click Customize to open the Symbol Style dialog box. You can use this dialog box to choose and format a symbol.
