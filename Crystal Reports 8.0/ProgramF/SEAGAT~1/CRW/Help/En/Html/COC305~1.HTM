Using the Set Print Date/Time command

  1. Choose the Set Print Date/Time command from the Report menu. The Set Print Date/Time dialog box appears.
  2. Click the Today's Date/Time option to print the current date. The current date is determined by the settings in the Date/Time Control Panel on your Windows system.
  3. Click the Other option to set the date that is printed by the Print Date field.
  4. Set the Year, Month, and Day by entering in a new value in the appropriate text box.
  5. Click OK. When the report is printed, the Print Date field will display the specified date.

Note:    Changes you make with this command will affect formulas with functions that contain reference to the current date (Aged30days, AllDatesToToday, and so on). When you change the print date, these functions do their calculations based on the new print date.
