Date / Time tab (Format Editor)

Use the Date / Time tab to set the formatting options for the date and time fields in your report.

Style (Date / Time tab/Format Editor)

From this pre-defined list, choose a date/time style to apply to the field selected on your report. When you choose System Default Long Format or System Default Short Format, the program uses the time and date formats from the Regional Settings Properties dialog box in the Windows Control Panel.

Note:    You can create a custom date/time format by clicking the Customize button. Your customized style appears in the style list as Custom Style.

Customize

Use this button to see more detailed formatting options on three tabs, called Data/Time, Date, and Time.
