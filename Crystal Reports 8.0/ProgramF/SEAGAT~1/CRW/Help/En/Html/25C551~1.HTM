Lotus Domino

Lotus Domino is a powerful groupware application used for communication and information sharing between departments in an organization.

Choosing Lotus Domino during installation adds the translation files P2snote.dll and P3slnen.dll to the Windows or Winnt Crystal directory.

Note:    

  • Nextpwd.dll and the Lotus API DLL (Lcppn201.dll) are installed in the Lotus Domino executable directory.
  • The following line is added to the Notes.ini file:

EXTMGR_ADDINS=extpwd

  • The Lotus Domino executable and data directories are added to the PATH.
