Using the Cut command

  1. Select the text or object you wish to cut.
  2. Choose the Cut command from the Edit menu. The program cuts the text/object and holds it in the Clipboard for later use (or until it is overwritten when you next use the cut or Copy command). Cut text or objects are deleted from your report.
