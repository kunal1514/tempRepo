Go To dialog box

This dialog box appears when you choose the Go To command from the Report menu.

Use the Go To dialog box to jump a particular page while previewing the report. You can jump to:

  • a specific page number
  • the first page of the report
  • the last page of the report.
Page

Click this option to jump to a specific page in your report. In the box, enter the number of the page that you want to jump to.

First page

Click this option to jump to the first page of the report.

Last page

Click this option to jump to the last page of the report.
