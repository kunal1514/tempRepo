Paragraph Formatting command

Use the Paragraph Formatting command to format paragraphs in a text object. For each paragraph you can set the font, tabs, and indents. In order to activate this command, you have to double-click the text object.

When you choose the Paragraph Formatting command the Format Editor appears with the Paragraph Formatting tab (Format Editor) selected.

Note:    

  • When you select a field, or select a text object by clicking it, you can move, or resize the object, or apply formatting that affects all the text in the object.
  • When you select a text object by double-clicking it, you can edit the text itself, and apply formatting to sections of the text.
