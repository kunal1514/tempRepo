Map Region Resolver dialog box

The Map Region Resolver dialog box opens when you choose the Resolve Mismatch command from the Analyzer menu, or from the map's right-click menu. Resolve Mismatch is available only when you are working with a map on the Analyzer tab.

Use this dialog box when the map you are working with uses a different geographic name from the one used in the database. For example, a map of Europe may use "United Kingdom" while the database uses the abbreviation "U.K." Until you resolve the data mismatch, the map will not be able to display any information for the selected geographic region.

Note:    The first time you choose this command while creating or editing a report, a processing dialog box appears briefly.

Assign this heading

This list displays all the geographic headings used in your database that do not match a default heading. Select items from this list (one at a time) and resolve the mismatch by choosing a default heading in the To this keyword list.

To this keyword

Use this list to select and assign a keyword from the default headings. When you have made a choice, the mismatched heading in the Assign this heading list is linked to the default heading you selected in the To this keyword list.

Assign

Click Assign to resolve the mismatch and save the link between the database heading and the default heading. The new assignment is shown in the list on the bottom part of the Map Region Resolver dialog box.

Remove

Click Remove to delete the link between the database heading and the default heading. After removing the link, the database heading is unresolved. You can make another link for it if you choose.
