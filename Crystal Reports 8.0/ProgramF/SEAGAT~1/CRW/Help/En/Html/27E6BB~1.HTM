Editing an existing dictionary

  1. Click the Open button on the Crystal Dictionaries toolbar.

    The File Open dialog box appears.

  2. Choose the existing dictionary file and click OK.

    The Crystal Dictionary Expert appears.

  3. Use the Expert to make changes to the dictionary file.
