Server type

Message Tracking Log

To log on to the Message Tracking Logs server type
  1. Open the Data Explorer dialog box by choosing:
      • Log On/Off Server from the File menu (when there is no report opened),
      • Log On/Off Server from the Database menu (when there is a report opened).
  2. Select Message Tracking Log from the More Data Sources folder.
  3. Double-click the folder or double-click Make a New Connection. The MS Exchange Message Tracking Log dialog box appears.
  4. Enter the path for, or browse for, the directory where the tracking logs are located. By default, Exchange Server will place these files in the \\<machine name>\<share name>\exchsrv\Tracking.log.
  5. Click OK.
  6. Enter the date range you'd like to see log entries for in the Enter Message Tracking Log Dates dialog box.
  7. Click OK.
To set the location of a table accessing a Message Tracking Log
  1. On the Database menu, click Set Location.
  2. When the Set Location dialog box appears, click Set Location.
  3. Find and highlight the new Message Tracking Log you want to use in the Data Explorer dialog box and click Set when done.

    Verify that the new table has been selected by reviewing the Table, Server Type, Server Name, Database (alias name displayed), and User ID in the Location section of the Set Location dialog box.

  4. Click Done after reviewing the table information.
