Editing groups

To edit a group
  1. On the Report menu, click Change Group Expert.
  2. In the Change Group dialog box, select the group you want to edit.
  3. Click Options.
  4. In the Change Group Options dialog box, edit the group as necessary.
  5. Click OK.

    The report reflects the changes you have made to the group.
