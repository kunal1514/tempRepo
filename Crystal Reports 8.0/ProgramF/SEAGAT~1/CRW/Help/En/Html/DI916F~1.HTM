Define Named Group dialog box

This dialog box appears when you click either the New button, or the Edit button on the Specified Order tab of the Insert Group dialog box.

When you use specified order grouping, you have to use the Define Named Group dialog box to create the groups and to specify the values in each when you group in a specific order.

Named Group

Type the name you want to use for the group into this box. By default the name will appear in the group header section.

For example, if you have a group that contains customers from Arizona, Washington, and California, you can call that group "Western Region" by typing that name in this box.

There is a tab for the named group in which you can specify the selection criterion. Select a type of selection criterion from the list. To add another selection criterion, click the New tab.

Browse Data

Click this button to browse the field data. You can select data from the dialog box that opens and click the Paste button to paste data to the Selection list. Click the Done button to return to the Define Named Group dialog box.

Tabs

The tabs display the fields whose values you are using to create the group. The tabs are all related to the same group, and each tab relates to a specific portion of the query used to create the group. When you click a tab, the appropriate portion of the query appears in the dialog box. If you want to expand the query, click the New tab.

  • To edit a named group, click the tab for that group.
  • To create a new group, click the New tab.
