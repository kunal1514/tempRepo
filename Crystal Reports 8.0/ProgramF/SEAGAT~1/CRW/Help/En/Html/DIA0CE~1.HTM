Links dialog box

This dialog box appears when you have a linked OLE object selected and you choose the Links command from the Edit menu.

Use the Links dialog box to update and modify existing object links in your report.

Links

Displays the current links used in the report, their type and the update status. To change the status of a link, click the link and then click Automatic or Manual.

Update

Specifies whether you want the selected link to be updated automatically or manually.

If you click Automatic, whenever you change the linked information in the original file, this report will be updated.

If you click Manual, you need to click Update Now when you want the information to be updated.

Update Now

Click this to update the information in the selected link.

Open Source

Click this to open the source file for the selected link.

Change Source

Click this to change the source file for the selected link.

Use this button to specify a new location for an object file, for example. When you click this button, the Change Source dialog box appears. Using this dialog box, select your new source. For example, if you have linked to Red.bmp and you have moved the bitmap from one directory to another, locate the new directory in this dialog box and select Red.bmp in that directory as your new source.

Break Link

Click this to disconnect the selected link. This is useful if you no longer want changes in the source file to be reflected in the current report. The object remains in the report as an unlinked, static object that cannot be edited via OLE.
