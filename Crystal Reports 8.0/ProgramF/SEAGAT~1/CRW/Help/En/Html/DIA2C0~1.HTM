Field options

Show Field Names

When you select this check box, field names are displayed for each report field. When you clear the check box, symbolic characters (XXXX, 555, and so on) are displayed instead of field names. This option is selected by default.

Insert Detail Field Titles

When you select this check box, a default field title appears in the Page Header section whenever you insert a field in the Details section of the Design tab. The default title is the name of the field, underlined. This option is selected by default.

Insert Group Name with Group

When you select the check box, the program automatically enters the Group Name in the Group Header whenever you insert a new group. This option is selected by default.

Preview pages

Starts With options

These three options let you specify which level of magnification you want the program to use as a default when displaying reports on the Preview tab. You have the following choices:

  • Full Size
    Displays the report at full size and fits as much of the report into the Preview tab as possible. This option is selected by default.
  • Fit Width
    Adjusts the width of the report to match the width of the Preview tab.
  • Fit Page
    Adjusts the width and the height, in order to be able to display the entire page in the Preview tab.

See also the Report Options dialog box.

Create Group Tree

When you select this check box, the Group Tree View is shown on the Preview tab. This option is selected by default. See also the Report Options dialog box.
