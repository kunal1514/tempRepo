Exporting to Lotus Domino

Seagate Crystal Reports enables you to export to a Lotus Domino client version 3.0 or greater.

Note:    The Lotus Domino OS/2 client is not supported.

To export to Lotus Domino
  1. Using the Lotus Domino Designer open an .nsf file and create a new form.
  2. Add the following optional fields to the form to provide the user with additional information:
      • Report - name of the report
      • Date - date and time of export
      • UserName - name of the user who exported the report
      • Comments - area where extra information can be added.
  1. Save the form as Report Form.
  2. Open the Seagate Crystal Report you would like to export.
  1. On the File menu, select Print, then click Export.

    The Export dialog box appears.

  2. Select the export format type from the Format drop-down list. In this case, choose Excel 7.0 (XLS) (Extended).
  3. Select Lotus Domino Database from the Destination drop-down list.
  4. Click OK.

    The Format Options dialog box appears.

  5. Change the format settings as required.
  6. Click OK.

    The Select Database dialog box appears.

  7. Double-click the Lotus Domino server to export the report to.
  8. Select the .nsf file to export the report to.
  9. Click OK.

    The Comments dialog box appears.

  10. Enter comments that are to appear when another user selects the report from the Lotus Domino Desktop.
  11. Click OK.

    The report appears as an attachment in a document within the .nsf file.

The next time a user logs onto the Lotus Domino database you specified, they will see the report in their desktop. The user can double-click the report file name to display the comments you wrote and then double-click the report icon to view the report.
