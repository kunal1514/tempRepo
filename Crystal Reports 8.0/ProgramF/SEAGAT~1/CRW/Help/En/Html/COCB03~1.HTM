Using the Paste command

To paste text within a text object
  1. Position the I-beam cursor at the place where you want the text to appear and click.
  2. Choose the Paste command from the Edit menu. The program pastes the text behind the text cursor.
To paste an object
  1. Choose the Paste command from the Edit menu.
  2. Drag the object outline to the place where you want the object to appear, and click to place the object.
