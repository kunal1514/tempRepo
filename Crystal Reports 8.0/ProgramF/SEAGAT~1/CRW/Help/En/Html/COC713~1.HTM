Using the Remove from Report command

  1. Choose the Remove from Report command from the Database menu. The Remove from Report dialog box appears. As you select a database from the list, the program displays the location (path) of that database in the Location box at the bottom of the dialog box.
  2. Select the database you want to delete, and click Remove.
  • If the report does not contain fields from this database, the program removes the database from the report.
  • If the report does contain fields from this database, the program prompts you to remove the fields from your report before it removes the database from the active list.

    Note:    The fields that need to be deleted can be found in the report itself, in the selection formula, in the sort order, or in a formula field. Check all possibilities.

  1. Click Done when finished.
