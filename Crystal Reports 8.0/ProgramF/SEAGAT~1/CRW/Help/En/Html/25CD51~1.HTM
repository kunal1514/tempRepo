DB2 Server

IBM's DB2 Server is an SQL compliant, client/server application that is part of IBM's popular database management suite.

Seagate Crystal Reports opens SQL data created by DB2 through the P2sdb2.dll, installed with Seagate Crystal Reports. This translation file works with the DB2 database drivers to read DB2 data. You must have the DB2 client installed and configured properly before using this translation file.
