Customize Map dialog box

The Customize Map dialog box appears when you choose Change Map Type from the Analyzer menu. You can also choose the option from the right-click shortcut menu on the Analyzer Tab. In either case, Change Map Type appears only when viewing a map on the Analyzer tab.

This dialog box allows you to change your map type and to set the options for the map type you select. The Customize Map dialog box contains much the same functionality as the Map Expert's Type Tab.

The Customize Map dialog box provides five basic map types (all types may not be available for every map), each suitable for a different strategy of data analysis. When deciding which map type best fits your report, you should consider the type of data you want to analyze. For example, if you want the map to display one data item for each geographic division (city, state, country, and so on), then you might use a Ranged, Dot Density, or Graduated map. On the other hand, if you want the map to display more than one value for each new geographic division, then you could use a Pie Chart map or a Bar Chart map.

Map Type

Use this list to select a map type and to activate the related mapping options.

The map types are:

  • Ranged
  • Dot Density
  • Graduated
  • Pie Chart
  • Bar Chart.
Options

Use the Options area to select the appearance of the map values displayed on the map. The options available in the Customize Map dialog box vary depending on the map layout you selected.

For more information about the options available for each map type, see:

  • Ranged Option view
  • Dot Density Option view
  • Graduated Option view
  • Pie Chart Option view
  • Bar Chart Option view.
