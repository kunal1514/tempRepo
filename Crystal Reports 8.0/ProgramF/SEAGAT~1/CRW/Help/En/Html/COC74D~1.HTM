Font command

Use the Font command to change the font face, size, style, effects, and/or color of selected text on your report.

  • When you choose the Font command after selecting a field, or after selecting a text object by clicking it, the Format Editor appears with the Font tab (Format Editor) selected. When a text object is selected in this way, you can move, resize the object as a whole, and any font settings affect all the text in the object.
  • When you choose the Font command after selecting a text object by double-clicking it, the Text Format dialog box appears. When a text object is selected in this way you can edit the text itself, and apply font settings to sections of the text.
