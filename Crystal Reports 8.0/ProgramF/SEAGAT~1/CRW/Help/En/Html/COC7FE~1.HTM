Using the Verify on Every Print command

The Verify on Every Print command is a lock that triggers the Verify Database command command every time you print in order to:

  • print with the current version of the active database
  • reset the record buffer to the current record size
  1. Choose the Verify on Every Print command from the Database menu.

    A check mark indicates that the command is active. By default, the command is inactive.

  2. Switch the check mark on/off as desired. The program will activate or deactivate the command accordingly.
