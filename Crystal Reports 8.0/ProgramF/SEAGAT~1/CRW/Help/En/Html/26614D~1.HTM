Adding fields to a query

  1. Click the field from the Database Fields area you would like to have appear in your query.
  2. Click Add to move it to the Query Fields area.
  3. Click Remove to remove a field from the Query Fields area.

Identifying unique values in a query

Select the Select Distinct Values check box to activate it.
