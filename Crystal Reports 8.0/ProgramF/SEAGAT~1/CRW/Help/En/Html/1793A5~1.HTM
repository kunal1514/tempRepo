Format Types

Seagate Crystal Reports provides you with nineteen different export format types. They include:

  • Character-separated values
  • Comma-separated values (CSV)
  • Crystal Reports (RPT)
  • Crystal Reports 7 (RPT)
  • Data Interchange Format (DIF)
  • Microsoft® Excel
  • HTML 4.0
  • HTML 3.2
  • Lotus 1-2-3
  • ODBC
  • Page Separated HTML
  • Paginated Text
  • Adobe Acrobat(PDF)
  • Record style (columns of values)
  • Report Definition
  • Rich Text Format
  • Tab-separated text
  • Tab-separated values
  • Text
  • Word for Windows document.

In addition to the standard export format types installed on your machine you may find additional export format types are available to you. These are determined by the DLL files on your local machine.

Note:    When you export a report to a file format other than Seagate Crystal Reports format (.RPT), you may lose some or all of the formatting that appears in your report. However, the program attempts to preserve as much formatting as the export format allows.
