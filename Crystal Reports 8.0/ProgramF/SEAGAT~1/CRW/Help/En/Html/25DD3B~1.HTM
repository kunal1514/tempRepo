Local file system

You can use Seagate Crystal Reports to report on files located on your system or network drives including information such as file name, version number, date, and so on.

The translation file P2sfs.dll is provided by Seagate Crystal Reports for reporting on your local file system.

Note:    You must have UNC (universal naming convention) read-access to your local file system. UNC is a standard format for paths that include a local area network file server. The basic syntax is: \\server\share\path\filename.
