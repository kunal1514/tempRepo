SAP

SAP, a vendor of client/server business application solutions, offers an ERP (enterprise resource planning) software package that includes SQL databases you can report on using Seagate Crystal Reports.

Seagate Crystal Reports provides access to databases created by SAP ERP. The P2srfc.dll and P2srfcaq.dll files installed with Seagate Crystal Reports translate requests to the SAP Application Server's network connectivity layer. The SAP Application Server then interprets the request and returns data from the SAP database.

Note:    Seagate Crystal Reports supports access to the SAP database using the files librfc32.dll and vrf32.dll.
