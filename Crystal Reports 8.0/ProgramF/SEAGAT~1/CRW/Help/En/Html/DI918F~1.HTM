Enter Parameter Values dialog box

This dialog box appears when you run a report that includes a parameter field, or when you refresh the data for that report.

Use the Enter Parameter Values dialog box to supply a parameter value if you want the program to run the report using a value other than the default value. This dialog box includes a list that included each parameter field in the report.

Parameter Fields

When the Enter Parameter Values dialog box appears, it displays a list of all the parameter fields in your report. You can select any parameter field in the list to set new value for it.

If prompting text was specified when the parameter field was created, that text appears below the Parameter Fields list on the Enter Parameter Values dialog box. The text changes as you select new parameter fields from the list.

Value options

The bottom part of the Enter Parameter Values dialog box contains a field or fields for you to enter the value(s) you want to use when running your report.

The fields that appear in this area vary depending on a number of things such as:

  • the data type of the field the parameter is based on (String, Currency, Boolean, Date/Time, and so on)
  • whether the parameter is range limited (in other words, if it allows you to enter a start and end value)
  • whether the parameter allows the use of multiple values.

For example, if you created a parameter based on a date field and allowed a range to be specified, the value options on the Enter Parameter Values dialog box would include:

  • Start of range
    select a value from a calendar to represent the first date in the range (including the date specified if you want)
  • End of range
    select a value from a calendar to represent the last date in the range (including the date specified if you want)
  • No lower Bound
    select this check box to specify a date range with no start date
  • No upper Bound
    select this check box to specify a date range with no end date.
