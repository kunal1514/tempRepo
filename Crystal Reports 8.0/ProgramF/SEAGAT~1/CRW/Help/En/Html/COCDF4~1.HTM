Select Expert command

Use the Select Expert to select the records or groups you want to include in your report, and to filter out those you don't want to include. For example, you may have a customer database that contains records for customers from every state and Canadian province, but you may want to do a report only on Texas customers. This command lets you restrict your report so that only Texas customers are included.

When you choose the Select Expert, if you have selected a field the Select Expert appears. If you have not selected field, you have to choose a field from the Choose Field dialog box before the Select Expert appears. Seagate Crystal Reports automatically generates a record or group selection formula based on settings you specify in the dialog box.

This command allows you to set up reasonably complex selection criteria, but to have more control over the formula that is generated. Use the Edit Selection Formula submenu on the Report menu and set up your selection formula using the Record Selection Formula Editor or Group Selection Formula Editor.

Note:    This command is also available as a button on the Standard toolbar.
