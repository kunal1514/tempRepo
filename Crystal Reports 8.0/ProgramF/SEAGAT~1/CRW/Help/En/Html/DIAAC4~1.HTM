Toolbars dialog box

Use the Toolbars command to show or hide toolbars. There are four toolbars to choose from: Standard, Formatting, Supplementary Tools, and Analyzer Tools. Use the check boxes provided to show any or all of the toolbars.

Large buttons

Select this option to view the buttons on the toolbar in a larger format and font.

Show ToolTips

Select this check box to have the program show tool tips for the buttons on the toolbar. Tool tips appear with the name of the button when you leave your mouse pointer over a button for a few seconds. This check box is selected by default.
