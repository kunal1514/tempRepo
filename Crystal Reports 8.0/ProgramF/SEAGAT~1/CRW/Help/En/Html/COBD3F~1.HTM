Object command

Note:    The name of this command and its submenu change to match the object you have selected.

Use the appropriate Object command, or the appropriate submenu command, to modify the selected embedded object.

  • Edit command
  • Open command
  • Properties command
  • Activate Contents command
  • Edit Package command
  • Convert command
  • Convert Picture Object command
