Cross-Tab option view

Use the Cross-Tab layout button when mapping on a cross-tab object. The cross-tab information can be used as the basis of the map. A Cross-Tab map does not require groups or summary fields. With the Cross-tab layout, you determine the Map on value, the Geographic field breakdown, and whether or not the data will be subdivided.

Geographic Field

Use the Show Geographic Field list to determine the geographic breakdown of your map. For example, if you select the Customer.Country field, your map will display values for each country. If you select the Customer.Region field, your map will display values for each region.

Subdivided by

Use the Subdivided by list to select an additional field to map on, if desired. The program uses this field to subdivide the pie or bar charts appearing on your map.

Map on

The Map on list displays the value options for your map. In other words, the data contained in the field in the Show list will be used by the program as your map values. If the Show list contains the Sum of Customer.Last Year's Sales field, your map values will be last year's sales for each customer.
