First Week of Year constants

The following constants are defined with the corresponding numeric values. You may use them with date functions that accept firstWeekOfYear as a parameter:

Constant Value Description

crUseSystem

0

Use the NLS API setting.

crFirstJan1

1

Start with week in which January 1 occurs (default).

crFirstFourDays

2

Start with the first week that has at least four days in the new year.

crFirstFullWeek

3

Start with first full week of the year.
