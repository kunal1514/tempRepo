Using the Report Viewer for HTML

Type the URL for the report into your web browser. Obtain this from your Systems Administrator or Information Systems Manager.

The first page of the requested report appears as a standard HTML page inside the Report Viewer.

In addition to the functions provided on the Standard toolbar, this Report Viewer has extra functions in its customized toolbar:

  • First page
  • Previous page
  • Next page
  • Last page
  • Goto page
  • Refresh report
  • Search in report.
