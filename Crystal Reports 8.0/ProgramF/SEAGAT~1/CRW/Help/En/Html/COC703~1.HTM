Using the Paragraph Formatting command

Use the Paragraph Formatting command to set alignment and indentation for a paragraph within a text object.

  1. Click within the text object that you want formatted and choose the Paragraph Formatting command from the Format menu. The Format Editor appears with the Paragraph Formatting tab (Format Editor) selected.
  2. Select the desired alignment from the Alignment list.
  3. Enter the desired indentations (in inches) into the appropriate edit boxes.
  4. Click OK.

Note:    The unit of measurement used in the dialog box (inches or centimeters) is based on the Measurement setting in the International section of the Windows Control Panel (English = inches, Metric = Centimeters).
