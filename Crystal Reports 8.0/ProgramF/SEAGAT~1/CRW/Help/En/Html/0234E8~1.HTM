Increased developer power

These enhancements are available in Seagate Crystal Reports Developer Edition.

Report Designer Component enhancements

Report Creation APIs

The RDC object model has been significantly enhanced to include the complete report creation API (runtime license required) to support building or modifying reports in applications at runtime.

Unbound Fields

Unbound fields, which bind to a database at runtime, are also supported and several new reporting processing events have been added. You can now insert an unbound field into a report at design time, and bind the field object to an actual data database field at runtime. This allows you to customize reports more easily at runtime.

Support for Microsoft Transaction Server (MTS)

The Report Designer Component now supports apartment model threading for better performance when used as a server or with MTS.

Web developer enhancements

Integrating Reports into Visual InterDev

The Report Integration Control enables web developers using Microsoft Visual InterDev 6.0 to write web applications that generate Crystal reports hosted on either a Web Component Server or an ASP server. End users of these applications can then view these reports using a Report Viewer running in a web browser.
