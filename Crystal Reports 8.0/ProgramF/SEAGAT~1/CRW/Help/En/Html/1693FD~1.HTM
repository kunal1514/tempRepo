To create an Informix MetaCube Cross-Tab report

  1. On the File menu, click New.

    The Seagate Crystal Report Gallery appears.

  2. Select the Cross-Tab Expert.

  3. Click OK.

    The Cross-Tab Report Expert appears.

  4. Click Database.

    The Data Explorer appears.

  5. Expand the More Data Sources folder, then expand the Informix Metacube folder.

  6. In the RDBMS Login & Choose DSS System dialog box, enter the password, then click OK.

    Note:    If you have already been working with Informix MetaCube reports during this session, you may not be required to login.

  7. Select the Audio Sales YTD by Channel and Brand Channel Net Profit tables, then click Add, then click Close.

    The Cross-Tab Report Expert appears with the Links tab active.

  8. Click Next.

    The Fields tab appears.

  9. Drag and drop the following fields from the Report Fields area to the appropriate Cross-Tab areas:
      • Attributes.Channel.Channel field from the Audio Sales YTD Channel table to the Rows area.
      • Attributes.Product.Brand field from the Brand Channel Net Profit table to the Columns area.
      • Measures.Sales Transactions.Net Profit field from the Brand Channel Net Profit table to the Summarized Fields area.

  10. Click Preview Report to see a Cross-Tab report based on the fields you selected.
  11. Save the report.

    This report is used in the adding a field to a Cross-Tab report tutorial.
