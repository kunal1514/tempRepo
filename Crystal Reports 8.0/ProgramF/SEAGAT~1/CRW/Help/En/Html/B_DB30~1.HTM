Seagate Software web site

To connect to the Seagate Software web site
  1. Click Help.
  2. Select Seagate Software on the Web.
  3. Click Seagate Software Home Page.

Or enter the following address into your web browser:

http://community.seagatesoftware.com

Note:    "community" replaces "www" in the web address.
