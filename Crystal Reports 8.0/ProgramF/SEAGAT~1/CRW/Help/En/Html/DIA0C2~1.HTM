Link Dimension dialog box

This dialog box appears when you click the Add (>) button in the Format OLAP Grid Object dialog box.

Use the Link Dimension dialog box to link a dimension to a relational field, either in the report or in a database.

Relational Fields

This list displays all of the relational fields available for use in your report.

Link Dimension to Relational Field(s)

This list displays all of the relational fields that the current dimension will be linked to.

Add (>)

Click this button to add a selected field from the Relational Fields list to the Link Dimension to Relational Fields(s) list.

Add All (>>)

Click this button to add all of the fields listed in the Relational Fields list to the Link Dimension to Relational Fields(s) list.

Remove (<)

Click this button to remove a selected field from the Link Dimension to Relational Fields(s) list.

Remove All (<<)

Click this button to remove all the fields from the Link Dimension to Relational Fields(s) list.

Order

Use the Order buttons to move a selected field up or down in the Link Dimension to Relational Fields(s) list. The dimension will be linked to the first relational field that is found. Fields that appear higher in the list will be searched for first.
