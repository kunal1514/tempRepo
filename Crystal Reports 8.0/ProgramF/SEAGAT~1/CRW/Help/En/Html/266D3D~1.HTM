Link information

To find out more about a link
  1. Click the link line.

    The line becomes highlighted, along with the fields on either end of the line.

  2. Click Options.

    The Link Options dialog box opens, displaying the link details.

  3. Click Cancel to return to the Create SQL Expert.
  4. Click Next.

    The Fields tab appears.
