Adding the Report Viewer to a web page

Microsoft's Internet Explorer web browser supports the OBJECT tag in HTML. This tag can be used to add the Report Viewer for ActiveX to a web page. Use code similar to the following:

<OBJECT ID="CRViewer" WIDTH=100% HEIGHT=95%
    CLASSID="CLSID:C4847596-972C-11D0-9567-00A0C9273C2A">
</OBJECT>

When you install the Web Component Server, the Report Viewer for ActiveX is installed under:

\Program Files\Seagate Software\Viewers\ActiveXViewer

Additionally, a virtual directory named:

/viewer

is set up on your web server, which points to:

\Program Files\Seagate Software\Viewers
