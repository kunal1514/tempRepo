Implementation overview

To implement a Web Component Server complete the following five processes:

  1. Install the Web Component Server.
  2. Configure NT Services.
  3. Ensure the installation is successful.
  4. Create a sample web site.
  5. Specify the Report Viewer.

Note:    The Configure NT Services and the Specify the Report Viewer processes are optional.
