Accessing the Crystal Report Web Components Manager

To access the Crystal Report Web Components Manager
  1. Locate the wcmadmin.exe file.
  2. Double-click cmadmin.exe to launch the application.

    The Crystal Report Web Components Manager application appears. It has four tabs:

      • Web Component Server
      • Cache Settings
      • Report Viewing
      • Page Server.
