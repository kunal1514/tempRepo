Section contents

The following is a short description of each chapter in this guide.

Welcome to Web Reporting Administration

This section provides an overview of the user's guide and online help. It also lists the command, button, and key conventions used in the user's guide.

Web Reporting Solution Overview

This section identifies how Seagate's web products work together with your existing web server. Information about new features and enhancements is also included.

Web Component Server Implementation

This section contains the procedural information required to install and configure the Web Component Server.

Web Component Server Administration

This section provides details about how to use the Crystal Web Components Manager application included with the Web Component Server. Using this application you can customize the settings of the Web Component Server.

Web Component Server Commands

This section provides examples of Web Component Server commands that can be appended to URL requests. These URL requests can be used to prompt users to enter security information or parameter field values.

Report Viewers

This section contains information about how to configure Report Viewers to effectively meet the needs of your users.

Product Support

This appendix lists methods and contact information for obtaining product support.

Seagate Software International Office Directory

This appendix provides contact numbers for Sales and Product Support for Seagate Software offices, worldwide.
