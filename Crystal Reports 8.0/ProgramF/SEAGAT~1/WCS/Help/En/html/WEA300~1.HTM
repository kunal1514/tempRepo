ActiveX and Java

This area contains check boxes that enable you to give users access to the following functions:

  • Print Report
  • Display logo
  • Zooming control
  • Display GroupTree
  • Allow export.

Select or clear the Display GroupTree check box to control whether or not a Group Tree is generated for Smart Navigation inside Report Viewers.

Note:    Generating a group tree for a report requires the Web Component Server to make an additional pass through the report data to create the Group Tree. This can cause response delays and requires additional system resources, especially if the report contains a large number of groups or multiple groups within groups.
