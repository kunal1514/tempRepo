Server Port

The Server Port area contains the port number that the Web Component Server uses to listen for requests from a Web Component Server connector.

Note:    If you make changes to the Server Port information stop and restart the NT Services for the web server and Web Component Server. For all other changes stop and restart the NT Services for the Web Component Server.
