ISAPI/NSAPI

By utilizing the API extensions exposed by Microsoft and Netscape web servers, wcsinsapi.dll produces a faster, more robust system for report delivery to the web server.

The ISAPI extension works with Microsoft Internet Information Server (IIS) version 2.0 and later, as well as the Personal Web Server for Windows NT Workstations.

The NSAPI programming interface is available on all Netscape web servers.
