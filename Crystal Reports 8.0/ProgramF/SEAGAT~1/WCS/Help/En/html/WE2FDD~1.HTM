Printer considerations

The Seagate Page Server settings may need to be changed if the web server does not have a local printer installed and you are using the local system (or other local user) account.

Local accounts, including the local system account do not support the use of a network printer device and as a result the Report Engine uses a different mode to generate the Encapsulated Page Files (EPFs). The quality of these EPFs may not be as good as what they could be with a specified printer driver.

To configure NT Services

The steps below provide you with information about how to configure NT Services to use a user account that has a printer driver installed.

This procedure is made up of two processes:

  • creating a user
  • changing the services.
