Configuring NT Services

You may need to make changes to the Seagate Page Server services if:

  • the Page Server accesses secured reports or reports that are not on the local machine
  • the Page Server accesses reports that use data sources that cannot be accessed through the local system account
  • the web server does not have a local printer device.
