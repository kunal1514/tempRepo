Adding the Viewer to a web page

As a Java applet, the Report Viewer can be added to a web page using the standard HTML tag APPLET. The name of the public class exposed by the applet is Report Viewer. Thus, the following code displays the Report Viewer for Java:

<APPLET CODE="ReportViewer.class"
CODEBASE="http://<domain>/viewer/JavaViewer"
WIDTH=600 HEIGHT=400>
</APPLET>

When you install Seagate Crystal Reports or the Crystal Web Component Server, the Java viewer is installed under:

\Program Files\Seagate Software\Viewers\JavaViewer

Additionally, a virtual directory named

/viewer

is set up on your web server, which points to the

\Program Files\Seagate Software\Viewers

The Report Viewer for Java provides several optional parameters to customize the look of the viewer and to control its functionality. Apply values to these parameters using the standard PARAM tag in your HTML code.
