Changing the services

  1. Open the Services window.

    If you are not sure how to do this, refer to Microsoft Windows NT documentation.

  2. Select the Seagate Page Server from the Services list, then click Startup.
  3. Ensure Automatic is selected as the Startup Type.
  4. In the Log On As area, select This Account.
  5. Click the Browse button next to the This Account field, then locate the user you just created (CRWEBUSER).
  6. Click Add, then click OK.

    The Service dialog box appears.

  7. Enter the correct password for the user in the appropriate text boxes.

  8. Click OK to the Services confirmation dialog box.
  9. Close the Services window.
