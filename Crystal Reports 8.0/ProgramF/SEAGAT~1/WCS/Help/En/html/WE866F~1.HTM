PromptOnRefresh# command

Specifies whether report should prompt for parameter field values when refreshed.

promptOnRefresh#=<value>
«<value> is 0 or 1.»

For example:

promptOnRefresh=1

Note:    Reports that have the promptOnRefresh# command applied do not have their pages shared. Caching will be by user.
