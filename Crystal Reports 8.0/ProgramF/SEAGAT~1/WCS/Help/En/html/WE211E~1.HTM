Installed files

The following is a list of primary files installed for the Web Component Server:

  • wcsinsapi.dll
  • wcscgi.exe
  • pagerserver.exe
  • wcs_xn_crimage.dll
  • webcompserver.exe
  • wcs_xn_reportviewer.dll.

Each of these files is installed by default into the following directory:

C:\Program Files\Seagate Software\WCS

Note:    Refer to the Seagate Crystal Reports online Developer Help for a complete list of files installed with the Web Component Server.
