Report Viewing tab

Use the Report Viewing tab to control what options are available to users when they view a report inside a browser. This tab is made up of three areas:

  • All viewers
  • ActiveX and Java
  • HTML Page.

Also included in this tab is the Java Plug-in button.

Note:    The following doesn't apply to Report Viewers that are accessed via an Applet or Object tag in an HTML page. In these cases, viewer options are set via parameter tags in the page.

Select and clear the check boxes to turn the options on and off.
