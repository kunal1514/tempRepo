GF command (Group Selection)

Specifies a group selection formula. This command is similar to the selection formulas command.

GF=<formula>
«<formula> is a selection formula in string format.»

For example:

GF= Sum({customer.Sales},{customer.Region})>10000
«Selects all groups in which the sum of all customer
sales in each region is greater than 10,000.»
