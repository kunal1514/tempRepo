Cache Maintenance

In the Cache Maintenance area, you can specify the Maximum Cache Size in KBytes and the Minutes between cache file cleanup.

Maximum Cache Size in KBytes

This is the maximum space, in kilobytes, that can be used on the Page Server system drive to cache report pages. If report requests begin to exceed this drive space, older pages are deleted (based on a LRU algorithm) from the cache until there is room for newer pages.

Note:    To apply your changes stop and restart the NT Services for the Page Server.
